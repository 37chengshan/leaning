# 电磁场与电磁波 1-5章 分章学习网站

## 打开方式
双击打开 `index.html`，从总入口进入各章。

## 目录结构
- `index.html`：总入口
- `chapters/ch1/index.html`：第1章 矢量分析基础
- `chapters/ch2/index.html`：第2章 静电场
- `chapters/ch3/index.html`：第3章 静电场边值问题
- `chapters/ch4/index.html`：第4章 恒定电流场
- `chapters/ch5/index.html`：第5章 恒定磁场
- `mindmaps/index.html`：1-5章总串联思维导图
- `mindmaps/ch1.html` 至 `mindmaps/ch5.html`：每章独立思维导图
- `assets/style.css`：统一样式

## GitHub Pages
建议在仓库 Settings → Pages 中选择：
- Source: Deploy from a branch
- Branch: main
- Folder: /root
