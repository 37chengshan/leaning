复习路线

1–5章学习路线总览

建议先用总导图建立框架，再按章节路线学习，最后用公式索引和题目索引复盘。

第1章｜矢量分析基础

本章是后面所有电场、磁场公式的数学语言。先理解场、矢量、∇、梯度、散度、旋度，后面才不会只是在背公式。

  1. 场与矢量 知道物理量如何分布在空间里。
  2. ∇ 算子 理解空间变化如何被数学表达。
  3. 梯度/散度/旋度 分别理解变化最快、源汇、旋转。
  4. 积分定理 把局部性质和整体性质联系起来。

第2章｜静电场

本章把第一章的数学工具用到电荷和电场中，核心是 E、φ、D、边界条件和静电能量。

  1. 电场强度 先理解电场是单位正电荷受到的力。
  2. 电位 再用标量 φ 简化静电场计算。
  3. 介质与导体 理解材料对电场的影响。
  4. 边界与能量 掌握分界面和储能问题。

第3章｜静电场边值问题

本章把第二章的电位 φ 作为主角，学习如何通过边界条件求空间电位。

  1. 从 E 转向 φ 用电位简化静电场求解。
  2. 判断方程 有源用泊松，无源用拉普拉斯。
  3. 识别边界条件 分清电位、法向导数和混合条件。
  4. 使用镜像法 用等效电荷处理特殊边界。
学习正文看导图

第4章｜恒定电流场

本章把电场放进导电介质里，研究稳态电流如何流动、如何满足连续性以及如何产生功率损耗。

  1. 电流密度 J 理解电流不只是总量，还可以是空间分布。
  2. 连续性 掌握 ∇·J=0 的物理意义。
  3. 边界条件 学会跨介质处理 J 和 E。
  4. 静电比拟 把静电场方法迁移到电流场。
学习正文看导图

第5章｜恒定磁场

本章研究恒定电流如何产生磁场，以及磁场在介质中如何表现、如何跨边界变化、如何对载流线圈产生转矩。

  1. 从电流到磁场 理解恒定电流是磁场旋度的来源。
  2. B 与 H 区分磁感应强度和磁场强度。
  3. 磁化 理解材料内部磁矩排列造成的宏观效应。
  4. 边界和转矩 掌握磁介质边界条件与线圈受力矩。

推荐顺序：第1章工具 → 第2章静电场 → 第3章边值求解 → 第4章恒定电流场 → 第5章恒定磁场。不要跳过第1章的散度、旋度、斯托克斯定理，否则后面公式会变成死记硬背。
