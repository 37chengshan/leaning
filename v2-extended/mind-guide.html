你应该怎么从 0 学这四个专题

不要直接背公式。正确顺序是先补电路语言，再进入动态电路、黑箱参数、运放分析、频率响应。

第0章：零基础预备：把电路语言补起来

如果你连电压、电流、节点、KCL、KVL 都不熟，后面的拉普拉斯、二端口、运放和频率响应都会变成机械背公式。所以先把电路语言补上。

  1. 电荷与电流
  2. 电压与接地
  3. 欧姆定律
  4. 串并联
进入

第1章：拉普拉斯变换：新手从时域到 s 域

拉普拉斯变换的价值是把电容、电感造成的微分方程变成代数方程。S1 里也把复频域下电路分析概括成：先按换路定则求初值，再列 KCL/KVL，再拉普拉斯变换求解，最后反变换。

  1. 时域与 s 域
  2. 单位阶跃/冲激
  3. 常用变换对
  4. 三大性质
进入

第2章：二端口网络：输入输出黑箱和矩阵参数

二端口网络用矩阵描述输入端和输出端之间的关系。它让你不用关心内部每根线，只看端口变量。

  1. 端口条件
  2. 端口变量
  3. 线性互易
  4. Y 参数
进入

第3章：运算放大器：虚短虚断 + KCL

运放题对新手最友好，因为它套路非常固定：先写虚断，再写虚短，最后在输入相关节点列 KCL。

  1. 符号
  2. 节点电压
  3. 理想模型
  4. 虚断
进入

第4章：频率响应：从阻抗到波特图

频率响应回答：同一个电路遇到不同频率信号时，输出会放大、衰减还是移相。

  1. 频率与角频率
  2. 网络函数
  3. 相量阻抗
  4. RC/RL 滤波
进入

总主线

  1. KCL/KVL 是列方程的底层语言。
  2. 电容电感让电路出现时间和频率问题。
  3. 拉普拉斯把时间微分变成 s 域代数。
  4. 二端口把复杂电路抽象成输入输出矩阵。
  5. 运放用虚短虚断加 KCL 解题。
  6. 频率响应研究不同频率下输出/输入如何变化。
