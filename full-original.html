1.1 场：空间中每一点都有一个物理量

概念入门

场不是一个单独的数，而是一整片空间里的分布规律。比如教室里每个位置都有温度，这就是温度场；空间中每一点都有电场强度，这就是电场。

标量场：T = T(x, y, z, t)
矢量场：A = A(x, y, z, t)
通俗解释

你可以把场想成一张“空间地图”：地图上每个位置都写着一个数，或者画着一个箭头。

符号拆解
  • T 表示温度这类只有大小的量。
  • A 表示矢量场，既有大小又有方向。
  • x、y、z 表示空间位置。
  • t 表示时间。
什么时候用

看到题目说“某区域内每一点都有某个物理量”时，就要想到场。

易错提醒

不要把场理解成某一个点的量。某一点的值只是场在该点的取值。

小题：判断温度、电场、速度哪个属于场？
  1. 温度如果随空间位置变化，可写成 T(x,y,z)，是标量场。
  2. 电场强度 E(x,y,z) 在每点有大小和方向，是矢量场。
  3. 流体速度 v(x,y,z) 在每点有速度大小和方向，也是矢量场。
答案整理：三者都可以构成场；温度是标量场，电场和速度是矢量场。

1.2 标量与矢量：只有大小，还是既有大小又有方向

基础坐标分量

标量只有大小，比如温度、电位；矢量既有大小又有方向，比如力、速度、电场强度、磁场强度。

A = Axex + Ayey + Azez
|A| = √(Ax2 + Ay2 + Az2)
通俗解释

一个空间箭头可以拆成三个方向上的分量。模长就是这个箭头的真实长度，本质上是三维勾股定理。

符号拆解
  • A 是矢量。
  • Aₓ、Aᵧ、A_z 是三个坐标方向的分量。
  • eₓ、eᵧ、e_z 是三个单位矢量。
  • |A| 是矢量大小。
什么时候用

给出矢量分量时，用这个式子表达矢量或求矢量大小。

易错提醒

不要把分量当成矢量本身。分量只是数，乘上单位矢量后才说明方向。

小题：A=3eₓ+4eᵧ，求 |A|。
  1. 识别分量：Aₓ=3，Aᵧ=4，A_z=0。
  2. 代入模长公式：|A|=√(3²+4²+0²)。
  3. 计算：√25=5。
答案整理：|A|=5。

1.3 点积与叉积：矢量相乘的两种方式

运算方向

点积衡量两个矢量同向程度，结果是标量；叉积衡量两个矢量构成的垂直效果，结果是矢量。

A·B = |A||B|cosθ = AxBx + AyBy + AzBz
|A×B| = |A||B|sinθ
通俗解释

点积像把一个矢量投影到另一个方向上；叉积像用右手定则判断旋转轴方向。

符号拆解
  • θ 是 A 和 B 的夹角。
  • cosθ 描述同向程度。
  • sinθ 描述垂直程度。
  • A×B 的方向由右手定则决定。
什么时候用

算功、通量、环量常用点积；算力矩、旋度方向、磁场力常用叉积。

易错提醒

点积结果不是矢量；叉积顺序不能随便换，A×B = −B×A。

小题：A=2eₓ+eᵧ，B=eₓ+3eᵧ，求 A·B。
  1. 写出分量：Aₓ=2，Aᵧ=1；Bₓ=1，Bᵧ=3。
  2. 用点积分量公式：A·B=AₓBₓ+AᵧBᵧ。
  3. 代入：2×1+1×3=5。
答案整理：A·B=5。

1.4 ∇ 算子：空间变化检测器

核心工具偏导

∇ 不是普通数字，而是由三个方向偏导数组成的微分算子。它负责检查一个场沿 x、y、z 三个方向怎么变化。

∇ = ex∂∂x + ey∂∂y + ez∂∂z
通俗解释

它像一个三方向扫描器：分别扫描 x、y、z 方向上的变化。后面接标量得到梯度，点乘矢量得到散度，叉乘矢量得到旋度。

符号拆解
  • ∂/∂x 表示只看 x 方向变化。
  • ∂/∂y 表示只看 y 方向变化。
  • ∂/∂z 表示只看 z 方向变化。
  • eₓ、eᵧ、e_z 标明方向。
什么时候用

遇到梯度、散度、旋度时，都先想到 ∇。

易错提醒

∇ 本身不是结果，它必须作用到某个标量场或矢量场上才有意义。

小题：∇φ、∇·A、∇×A 分别叫什么？
  1. ∇ 作用于标量 φ，得到 ∇φ，叫梯度。
  2. ∇ 与矢量 A 做点积，得到 ∇·A，叫散度。
  3. ∇ 与矢量 A 做叉积，得到 ∇×A，叫旋度。
答案整理：∇φ 是梯度，∇·A 是散度，∇×A 是旋度。

1.5 梯度与方向导数：标量场变化最快的方向

梯度方向导数

梯度用于标量场，告诉你该标量在哪个方向增加最快，以及增加得有多快。方向导数则是问沿某个指定方向变化多快。

∇φ = ex∂φ∂x + ey∂φ∂y + ez∂φ∂z
∂φ∂l = ∇φ · el
通俗解释

梯度像山坡上最陡的上坡方向。方向导数像你不一定沿最陡方向走，而是沿题目指定方向走，看看高度变化多快。

符号拆解
  • φ 是标量场。
  • ∇φ 是梯度矢量。
  • e_l 是指定方向的单位矢量。
  • ∂φ/∂l 是沿 l 方向变化率。
什么时候用

求温度、电位、高度等标量场变化方向时用梯度；题目问“沿某方向变化率”时用方向导数。

易错提醒

方向向量必须先单位化，否则方向导数会被长度放大。

小题：φ=x²+y²，求点(1,2)处梯度。
  1. 求偏导：∂φ/∂x=2x，∂φ/∂y=2y。
  2. 代入点(1,2)：∂φ/∂x=2，∂φ/∂y=4。
  3. 写成矢量：∇φ=2eₓ+4eᵧ。
答案整理：∇φ|(1,2)=2eₓ+4eᵧ。

1.6 散度与通量：有没有源、有没有净流出

散度通量

散度是局部概念，判断某点附近是否像源一样向外发散；通量是整体概念，计算穿过一个面的总量。

∇·A = ∂Ax∂x + ∂Ay∂y + ∂Az∂z
Φ = ∬S A·dS
通俗解释

散度像看一个点是不是喷泉或排水口。通量像看一张网有多少水流穿过去。

符号拆解
  • A 是矢量场。
  • Aₓ、Aᵧ、A_z 是分量。
  • S 是曲面。
  • dS 是带方向的小面积。
什么时候用

判断场是否有源时用散度；求穿过某个面或闭合面的总量时用通量。

易错提醒

通量为零不一定说明没有场，可能是进出抵消。

小题：A=xeₓ+yeᵧ+ze_z，求散度。
  1. 识别分量：Aₓ=x，Aᵧ=y，A_z=z。
  2. 逐项求偏导：∂Aₓ/∂x=1，∂Aᵧ/∂y=1，∂A_z/∂z=1。
  3. 相加：∇·A=3。
答案整理：散度为 3，表示该场处处有向外发散趋势。

1.7 旋度与环量：有没有局部打转

旋度环量

旋度判断矢量场局部有没有旋转趋势；环量计算沿闭合路径走一圈的总绕行效果。

∇×A = (∂Az∂y − ∂Ay∂z)ex + (∂Ax∂z − ∂Az∂x)ey + (∂Ay∂x − ∂Ax∂y)ez
Γ = ∮LA·dl
通俗解释

把小叶轮放进场里，如果叶轮会转，说明旋度不为零。环量则是沿一整条闭合曲线累加这个绕行效果。

符号拆解
  • ∇×A 是旋度，结果是矢量。
  • Γ 是环量。
  • L 是闭合曲线。
  • dl 是沿路径方向的小线段。
什么时候用

判断场是否无旋、是否能写成某个势函数梯度时用旋度。

易错提醒

旋度公式正负号容易写反，建议按行列式顺序记。

小题：A=−yeₓ+xeᵧ，求旋度 z 分量。
  1. 识别 Aₓ=−y，Aᵧ=x。
  2. z 分量公式：(∇×A)_z=∂Aᵧ/∂x−∂Aₓ/∂y。
  3. 计算：∂x/∂x=1，∂(−y)/∂y=−1。
  4. 相减：1−(−1)=2。
答案整理：(∇×A)_z=2，说明绕 z 轴有旋转趋势。

1.8 散度定理与斯托克斯定理：局部和整体互相转换

积分定理高斯定理

散度定理把体内散度与边界通量联系起来；斯托克斯定理把曲面上的旋度与边界环量联系起来。

∯SA·dS = ∭V(∇·A)dV
∮LA·dl = ∬S(∇×A)·dS
通俗解释

第一条说：内部所有源的总和等于边界总流出。第二条说：边界绕行效果等于内部所有局部旋转效果的总和。

符号拆解
  • S 是闭合曲面。
  • V 是曲面包围的体积。
  • L 是曲面的边界曲线。
  • dV 是体积元。
什么时候用

复杂面积分和体积分之间转换时使用；后面高斯定律、安培环路定理都会用到这些思想。

易错提醒

散度定理要求 S 是闭合曲面；斯托克斯定理的 L 是曲面 S 的边界。

小题：闭合面通量为正说明什么？
  1. 通量是穿过闭合面的净流出量。
  2. 通量为正，说明流出多于流入。
  3. 用散度定理看，内部散度积分为正，表示内部存在净源。
答案整理：闭合面内存在净源，整体向外流出。

2.1 电场强度 E：单位正电荷受到的力

静电场定义

电场强度描述某点电场有多强、方向朝哪里。它的方向规定为正电荷在该点受到的力的方向。

E = Fq
通俗解释

把一个很小的正试探电荷放到某点，看它受到多少力。每 1 C 电荷受到的力，就是该点电场强度。

符号拆解
  • E 是电场强度，单位 N/C 或 V/m。
  • F 是电场力。
  • q 是试探电荷量。
什么时候用

题目给电场力和试探电荷时，用定义式。

易错提醒

如果试探电荷是负电荷，受力方向和电场方向相反。

小题：q=2 μC，受力 6 mN，求 E。
  1. 统一单位：q=2×10⁻⁶ C，F=6×10⁻³ N。
  2. 代入 E=F/q。
  3. E=(6×10⁻³)/(2×10⁻⁶)=3×10³ N/C。
  4. 若 q 为正，E 方向与 F 相同。
答案整理：E=3×10³ N/C，方向与正电荷受力方向一致。

2.2 电场线：用图像看方向和强弱

电场线直观理解

电场线是人为画出的辅助线。切线方向表示电场方向，疏密表示电场强弱。

dxEx = dyEy = dzEz
通俗解释

这表示电场线在任一点的前进方向，都和该点电场矢量方向一致。

符号拆解
  • dx、dy、dz 是电场线小位移分量。
  • Eₓ、Eᵧ、E_z 是电场分量。
  • 比例相等说明线的切向与 E 平行。
什么时候用

判断电场线方向、画电场分布示意图时使用。

易错提醒

电场线不是实际存在的绳子，它只是辅助表示。

小题：电场线越密代表什么？
  1. 电场线疏密用来表示电场强弱。
  2. 越密说明单位面积穿过的电场线越多。
  3. 因此该区域电场强度较大。
答案整理：电场线越密，电场越强。

2.3 电位 φ 与电场 E：电场指向电位降低最快方向

电位负梯度

静电场中可以用标量电位 φ 描述电场。电场强度等于电位梯度的负值。

E = −∇φ
φA − φB = ∫ABE·dl
通俗解释

电位像高度，正电荷会沿电位降低方向运动，所以电场方向是电位下降最快方向。

符号拆解
  • φ 是电位，单位 V。
  • ∇φ 指向电位增加最快方向。
  • 负号说明 E 指向电位降低方向。
  • E·dl 表示沿路径方向的电场贡献。
什么时候用

已知电位求电场时，用 E=−∇φ；求两点电位差时用线积分。

易错提醒

最常见错误是漏掉负号。

小题：φ=3x，求 E。
  1. 求梯度：∇φ=3eₓ。
  2. 用 E=−∇φ。
  3. 得到 E=−3eₓ。
答案整理：E=−3eₓ V/m。

2.4 真空中静电场方程：有源无旋

高斯定律无旋

静电场有两个基本性质：电荷是电场的源，所以散度与电荷密度有关；静电场无旋，所以可以引入电位。

∇·E = ρε₀
∇×E = 0
∯SE·dS = Qε₀
∮LE·dl = 0
通俗解释

∇·E 表示电场线发散程度；∇×E=0 表示静电场没有旋涡，沿闭合路径一圈做功为零。

符号拆解
  • ρ 是体电荷密度。
  • ε₀ 是真空介电常数。
  • Q 是闭合面内总电荷。
  • S 是闭合曲面，L 是闭合路径。
什么时候用

高斯定律适合对称电荷分布求 E；无旋性质用于引入电位。

易错提醒

高斯定律总成立，但不一定总方便求解。

小题：某处 ρ>0，∇·E 是正还是负？
  1. 根据 ∇·E=ρ/ε₀。
  2. ε₀ 为正。
  3. ρ>0，所以 ∇·E>0。
答案整理：∇·E 为正，说明该处电场线向外发散。

2.5 电介质中的 D：把自由电荷单独拿出来处理

电介质电位移

介质会被电场极化。引入 D 的目的，是在介质中更方便地处理自由电荷。

D = εE = εrε₀E
∇·D = ρf
∯SD·dS = Qf
通俗解释

E 描述电场强度；D 是为了处理介质中自由电荷而引入的辅助矢量。

符号拆解
  • D 是电位移矢量，单位 C/m²。
  • ε 是介电常数。
  • εᵣ 是相对介电常数。
  • ρ_f 是自由电荷密度。
什么时候用

介质中的高斯定律通常先求 D，再用 D=εE 求 E。

易错提醒

不要把自由电荷和束缚电荷混为一谈。

小题：线性介质中 D=4ε₀E，求 εᵣ。
  1. 线性介质关系：D=εᵣε₀E。
  2. 与题中 D=4ε₀E 比较。
  3. 得到 εᵣ=4。
答案整理：εᵣ=4。

2.6 导体静电平衡：内部 E=0，导体是等位体

导体等位体

导体中自由电荷能移动。静电平衡时如果内部仍有电场，电荷会继续移动，所以平衡条件是导体内部电场为零。

Einside = 0
φ = constant
通俗解释

E=−∇φ。若 E=0，则 ∇φ=0，说明导体内部电位不随位置变化，所以导体是等位体。

符号拆解
  • E_inside 是导体内部电场。
  • φ=constant 表示电位处处相同。
什么时候用

判断导体静电平衡、导体表面电荷分布时使用。

易错提醒

导体内部 E=0 不代表导体表面没有电荷，多余电荷通常在表面。

小题：静电平衡导体内部两点电位差是多少？
  1. 导体内部 E=0。
  2. 电位差与电场线积分有关。
  3. E=0，所以积分为 0。
答案整理：导体内部任意两点电位差为 0。

2.7 静电场边界条件：切向看 E，法向看 D

边界条件介质界面

场穿过不同介质分界面时，要分解成切向和法向。静电场中切向 E 连续，法向 D 的跳变由自由面电荷决定。

E1t = E2t
D2n − D1n = ρs
导体表面：Et=0，Dn=ρs
通俗解释

切向方向沿界面，法向方向垂直界面。边界条件就是告诉你场跨过界面时哪些分量不能乱跳。

符号拆解
  • t 表示切向。
  • n 表示法向。
  • ρ_s 是自由面电荷密度。
  • D=εE 可把 D 条件转成 E 条件。
什么时候用

介质分层、电场折射、导体表面电场题都要用。

易错提醒

不要把 E_t 连续误写成 D_t 连续；也不要把 D_n 条件直接当 E_n 连续。

小题：无自由面电荷时，D₁n 与 D₂n 关系？
  1. 边界条件：D₂n−D₁n=ρ_s。
  2. 无自由面电荷：ρ_s=0。
  3. 所以 D₂n−D₁n=0。
答案整理：D₁n = D₂n。

2.8 静电场能量：电场本身储存能量

能量电容

电场可以对电荷做功，所以电场本身具有能量。能量密度描述单位体积储存多少能量。

we = 12E·D
We = 12∭VE·D dV
W = 12CU2 = 12QU = Q22C
通俗解释

能量密度是局部量，总能量是把所有小体积里的能量加起来。电容器公式是常用特例。

符号拆解
  • w_e 是能量密度。
  • W_e 是总能量。
  • C 是电容。
  • U 是电压，Q 是电荷量。
什么时候用

计算电场储能、电容器能量时使用。

易错提醒

公式选择看题目给什么量：给 C/U 用 1/2CU²，给 Q/C 用 Q²/2C。

小题：C=2 μF，U=10 V，求电容器能量。
  1. 统一单位：C=2×10⁻⁶ F。
  2. 选公式 W=1/2CU²。
  3. 代入：W=1/2×2×10⁻⁶×10²。
  4. 计算：W=1×10⁻⁴ J。
答案整理：W=1.0×10⁻⁴ J。

3.1 静电场边值问题：已知边界，求空间电位

边值问题电位

边值问题就是：边界上的电位或电位导数已知，要求区域内部每一点的电位分布。

已知边界条件 → 求 φ(x,y,z) → 再由 E = −∇φ 求电场
通俗解释

它像知道房间墙壁温度，求房间内部温度分布；静电场中则是知道边界电位，求空间电位。

符号拆解
  • φ 是待求电位。
  • 边界条件是求解所需的约束。
  • E=−∇φ 用于由电位求电场。
什么时候用

遇到导体边界、电位已知、空间区域待求时使用边值问题思想。

易错提醒

边值问题不是只求某一点，而是求整个区域的函数分布。

小题：已知导体球表面电位，求球外电位属于什么问题？
  1. 题目给了边界上的电位。
  2. 要求空间区域中的电位。
  3. 这符合边值问题定义。
答案整理：属于静电场边值问题，具体是第一类边界条件问题。

3.2 泊松方程与拉普拉斯方程

微分方程电位

电位 φ 满足的方程取决于区域内有没有电荷。有电荷源时是泊松方程；无电荷区域是拉普拉斯方程。

泊松方程：∇²φ = −ρε
拉普拉斯方程：∇²φ = 0
通俗解释

ρ 是电荷源。如果区域里有电荷，电位会被源项驱动；如果没有电荷，电位只由边界决定。

符号拆解
  • ∇² 是拉普拉斯算子。
  • ρ 是体电荷密度。
  • ε 是介电常数。
  • φ 是电位。
什么时候用

判断区域有源或无源时选择对应方程。

易错提醒

不要把“空间里有电场”和“区域内有电荷”混淆。无源区也可以有电场。

小题：无电荷区域中电位满足什么方程？
  1. 无电荷说明 ρ=0。
  2. 泊松方程 ∇²φ=−ρ/ε。
  3. 代入 ρ=0，得到 ∇²φ=0。
答案整理：满足拉普拉斯方程 ∇²φ=0。

3.3 三类边界条件：狄里赫利、诺依曼、混合

边界条件分类

边界条件告诉你边界上已知什么。已知电位是第一类；已知法向导数是第二类；已知二者组合是第三类。

第一类：φ|S=f₁(r)
第二类：∂φ∂n|S=f₂(r)
第三类：aφ + b∂φ∂n = f₃(r)
通俗解释

第一类像告诉你海拔高度；第二类像告诉你坡度；第三类是高度和坡度的组合约束。

符号拆解
  • S 是边界。
  • ∂φ/∂n 是沿法线方向的变化率。
  • f₁、f₂、f₃ 是边界上给定函数。
  • a、b 是系数。
什么时候用

读题时先判断给的是电位、场的法向分量，还是二者组合。

易错提醒

法向导数和电场法向有关，因为 E_n=−∂φ/∂n。

小题：导体表面电位给定为 V₀，属于哪类边界条件？
  1. 题目直接给出边界电位。
  2. 第一类边界条件定义为 φ|S=f。
  3. 因此属于狄里赫利条件。
答案整理：属于第一类边界条件，也叫狄里赫利条件。

3.4 唯一性定理：解只要满足方程和边界，就不会错

唯一性理论基础

唯一性定理说明，在给定区域内，只要电位满足泊松/拉普拉斯方程，并满足边界条件，那么解就是唯一的。

方程 + 边界条件 → 唯一电位分布
通俗解释

这给镜像法等方法提供了底气：不管你怎么构造，只要边界和区域方程对，结果就是原问题的解。

符号拆解
  • 方程描述区域内部规律。
  • 边界条件描述区域边缘约束。
  • 唯一性说明不会存在两个不同正确解。
什么时候用

证明镜像法、判断构造解是否正确时使用。

易错提醒

唯一性定理不一定告诉你怎么找解，但告诉你找到后为什么可信。

小题：为什么镜像法只需检查边界条件？
  1. 镜像法构造的新电荷只放在求解区域外。
  2. 求解区域内仍满足原来的方程。
  3. 如果边界条件也相同，根据唯一性定理，区域内解相同。
答案整理：因为唯一性定理保证满足同一方程和同一边界条件的解唯一。

3.5 镜像法：用等效电荷代替导体边界影响

镜像法技巧

镜像法把复杂导体边界的作用，用边界外的等效电荷代替，使原问题变成无界空间中的电荷叠加问题。

原电荷 + 镜像电荷 → 满足边界条件的等效问题
通俗解释

镜像电荷不是真实存在的，它只是数学替身。它的作用是让导体表面电位条件被自动满足。

符号拆解
  • 原电荷是真实电荷。
  • 镜像电荷是构造的等效电荷。
  • 边界条件是镜像法成功的关键。
什么时候用

点电荷与无限大接地导体平面、导体球、导体圆柱等问题常用。

易错提醒

镜像电荷只能用于求解区域外或导体内部的数学构造，不能把它当真实电荷。

小题：点电荷 q 距接地无限导体平面 a，镜像电荷怎么放？
  1. 接地平面要求平面上电位为 0。
  2. 在平面对称位置放一个 −q。
  3. 原电荷和平面另一侧镜像电荷在平面上产生等大反号电位。
答案整理：在平面另一侧对称位置放镜像电荷 −q。

4.1 恒定电流场：电荷持续定向运动形成电流

恒定电流电流密度

恒定电流场研究稳态导电介质中的电流分布。稳态意味着各处电流不随时间改变。

J = σE
I = ∬SJ·dS
通俗解释

J 描述单位面积上通过多少电流；σE 说明导电材料中电场推动电荷形成电流。

符号拆解
  • J 是电流密度，单位 A/m²。
  • σ 是电导率。
  • E 是电场强度。
  • I 是通过截面的总电流。
什么时候用

求导体或导电介质中电流、电流密度时使用。

易错提醒

J 是面密度意义的矢量，不等同于总电流 I。

小题：σ=2 S/m，E=5 V/m，求 J。
  1. 使用欧姆定律微分形式 J=σE。
  2. 代入 σ=2，E=5。
  3. J=10 A/m²，方向与 E 相同。
答案整理：J=10 A/m²。

4.2 恒定电流场方程：电流连续且无旋电场

场方程连续性

稳态时没有电荷持续堆积，所以电流密度散度为零；恒定电场仍满足静电型无旋条件。

∇·J = 0
∯SJ·dS = 0
∇×E = 0
∮LE·dl = 0
通俗解释

∇·J=0 表示电流线没有起点和终点，不会在某处凭空产生或消失。

符号拆解
  • J 是电流密度。
  • E 是电场强度。
  • S 是闭合面。
  • L 是闭合曲线。
什么时候用

判断稳态电流分布和建立边界条件时使用。

易错提醒

∇·J=0 不是说 J=0，而是说流入和流出平衡。

小题：闭合面内恒定电流净流出为多少？
  1. 恒定电流满足 ∯J·dS=0。
  2. 这表示任意闭合面净电流为零。
  3. 所以流入等于流出。
答案整理：净流出为 0。

4.3 电流连续性原理：电流线必须闭合

连续性物理意义

恒定电流线没有起点和终点，必须连续闭合。这和静电场线不同，静电场线可以起于正电荷、止于负电荷。

稳态：∇·J = 0
通俗解释

如果某处电流线突然终止，就意味着电荷在那里不断堆积，这就不是恒定状态。

符号拆解
  • ∇·J 表示电流密度的散度。
  • 等于 0 表示没有净源汇。
什么时候用

解释导电介质中电流分布、边界法向电流连续时使用。

易错提醒

不要把电流线和电场线混同。电流线在稳态中闭合，静电场线可起止于电荷。

小题：为什么稳态导线中电流处处相等？
  1. 若某截面电流大、另一截面小，中间会出现电荷堆积或减少。
  2. 稳态不允许持续堆积。
  3. 所以沿串联导线路径电流连续。
答案整理：因为恒定电流满足连续性，不能在导线中途凭空产生或消失。

4.4 恒定电流场边界条件

边界条件导电介质

两种导电介质交界面上，法向电流密度连续，切向电场强度连续。

J1n = J2n
E1t = E2t
通俗解释

法向 J 连续保证电流不在界面堆积；切向 E 连续来自恒定电场的无旋性。

符号拆解
  • J₁n、J₂n 是界面两侧法向电流密度。
  • E₁t、E₂t 是界面两侧切向电场。
  • n 是法向，t 是切向。
什么时候用

分层导电介质、电流折射问题使用。

易错提醒

法向连续的是 J，不一定是 E；因为 J=σE，σ 不同时 E_n 会改变。

小题：σ₁≠σ₂ 且 J₁n=J₂n，E₁n 与 E₂n 一定相等吗？
  1. 由 J=σE 得 J_n=σE_n。
  2. 边界条件为 σ₁E₁n=σ₂E₂n。
  3. 若 σ₁≠σ₂，则 E₁n 一般不等于 E₂n。
答案整理：不一定相等。

4.5 导电介质功率损耗

功率焦耳热

电流在导电介质中流动会产生热损耗。单位体积损耗由 J·E 描述。

p = J·E = σE² = J²σ
P = ∭Vp dV
通俗解释

电场推动电荷运动，介质阻碍运动，这部分能量转化为热。

符号拆解
  • p 是单位体积功率损耗。
  • P 是总功率损耗。
  • σ 是电导率。
  • V 是导电区域体积。
什么时候用

求导体发热、介质损耗时使用。

易错提醒

注意 p 是功率密度，P 才是总功率。

小题：σ=4 S/m，E=3 V/m，求 p。
  1. 用 p=σE²。
  2. 代入 σ=4，E=3。
  3. p=4×9=36 W/m³。
答案整理：p=36 W/m³。

4.6 静电比拟：恒定电场与静电场的相似性

比拟电导电容

均匀导电媒质中的恒定电场，与均匀介质无源区中的静电场形式相似，可以用静电问题的结果类比电流问题。

σ ↔ ε
J ↔ D
G ↔ C
G = (σ/ε)C
通俗解释

把电导率 σ 看成介电常数 ε 的对应物，把电流密度 J 看成电位移 D 的对应物，就能迁移解法。

符号拆解
  • σ 是电导率。
  • ε 是介电常数。
  • G 是电导。
  • C 是电容。
什么时候用

几何形状相同、边界条件相同时，用已知电容推电导。

易错提醒

比拟要求边界条件和几何结构对应，不能随意套。

小题：某结构 C=2 pF，σ/ε=10⁶ s⁻¹，求 G。
  1. 用 G=(σ/ε)C。
  2. C=2×10⁻¹² F。
  3. G=10⁶×2×10⁻¹²=2×10⁻⁶ S。
答案整理：G=2 μS。

4.7 理想导电体中不存在恒定电场

理想导体极限思想

理想导电体电导率 σ→∞。若有限电流密度 J 存在，由 J=σE 可知 E 必须趋于 0。

E = Jσ，当 σ→∞ 且 J 有限时，E→0
通俗解释

理想导体几乎不需要电场就能维持电流，因此内部恒定电场为零。

符号拆解
  • σ 是电导率。
  • J 是电流密度。
  • E 是电场强度。
什么时候用

判断理想导体、电磁边界理想化时使用。

易错提醒

现实导体不是 σ 无限大，实际导线内部可以有很小电场。

小题：理想导体中若 J 有限，E 为多少？
  1. 由 J=σE 得 E=J/σ。
  2. 理想导体 σ→∞。
  3. 有限 J 除以无限大趋于 0。
答案整理：E=0。

5.1 真空中恒定磁场方程：有旋无散

磁场安培环路定理

恒定磁场由恒定电流产生。磁场没有磁单极子，所以 B 的散度为零；电流会激发磁场旋度。

∇×B = μ₀J
∇·B = 0
∮LB·dl = μ₀I
通俗解释

电流像磁场的旋涡源；磁感应线没有头尾，总是闭合的。

符号拆解
  • B 是磁感应强度。
  • J 是电流密度。
  • μ₀ 是真空磁导率。
  • I 是闭合曲线包围的电流。
什么时候用

分析真空中电流产生磁场时使用。

易错提醒

磁场线没有起点和终点，不像静电场线起于正电荷止于负电荷。

小题：为什么 ∇·B=0 表示磁力线闭合？
  1. 散度表示源汇。
  2. ∇·B=0 表示磁场没有源或汇。
  3. 没有源汇的场线不能凭空开始或结束，只能闭合或延伸到无穷远。
答案整理：表示磁场无源，磁力线无头无尾。

5.2 介质中的恒定磁场方程：用 H 处理传导电流

H介质磁场

在磁介质中，用磁场强度 H 表述安培环路定理，右边只考虑传导电流。

∇×H = J
∇·B = 0
∮LH·dl = I
通俗解释

H 的引入类似 D：把介质磁化影响从源项中分离出来，让方程右边主要看传导电流。

符号拆解
  • H 是磁场强度，单位 A/m。
  • B 是磁感应强度，单位 T。
  • J 是传导电流密度。
  • I 是传导电流。
什么时候用

磁介质中使用安培环路定理时常用 H。

易错提醒

不要把 B 和 H 混为一谈，它们通过介质本构关系联系。

小题：若闭合路径包围电流 5 A，则 ∮H·dl 等于多少？
  1. 介质中安培环路定理：∮H·dl=I。
  2. 题中 I=5 A。
  3. 直接得到结果。
答案整理：∮H·dl=5 A。

5.3 磁介质本构关系：B、H、M 的联系

磁化本构关系

磁介质在外磁场作用下会磁化。磁化强度 M 描述单位体积内磁矩的合成效果。

B = μ₀(H + M)
线性各向同性介质：B = μH = μrμ₀H
通俗解释

H 描述由传导电流激发的磁场强度，M 描述材料被磁化后的贡献，B 是综合后的磁感应强度。

符号拆解
  • M 是磁化强度。
  • μ 是磁导率。
  • μᵣ 是相对磁导率。
  • μ₀ 是真空磁导率。
什么时候用

处理磁介质、铁磁材料、磁屏蔽时使用。

易错提醒

线性关系 B=μH 不适用于所有强非线性铁磁材料的全过程。

小题：μᵣ=100，H=10 A/m，求 B 大小。
  1. 用 B=μᵣμ₀H。
  2. 代入 μᵣ=100，H=10。
  3. B=1000μ₀ T。
  4. 若 μ₀=4π×10⁻⁷，则 B≈1.26×10⁻³ T。
答案整理：B=1000μ₀ T≈1.26 mT。

5.4 磁化电流：磁化也等效为电流

磁化电流M

材料内部微观电流环在宏观上产生磁化效果，可以等效为磁化电流体密度和面密度。

Jm = ∇×M
Km = M×n
通俗解释

如果 M 在空间中有旋转变化，会形成体磁化电流；如果磁化介质有表面，会出现面磁化电流。

符号拆解
  • J_m 是磁化电流体密度。
  • K_m 是磁化电流面密度。
  • M 是磁化强度。
  • n 是表面法向单位矢量。
什么时候用

分析磁化介质等效电流时使用。

易错提醒

磁化电流不是外接电源提供的传导电流，而是材料内部磁化的等效描述。

小题：均匀 M 且无空间变化时，J_m 如何？
  1. J_m=∇×M。
  2. 若 M 均匀不随空间变化，各偏导为 0。
  3. 因此旋度为 0。
答案整理：J_m=0，但表面仍可能有 K_m。

5.5 恒定磁场边界条件

边界条件磁介质

两种磁介质分界面上，B 的法向分量连续；H 的切向分量是否跳变取决于界面传导面电流。

B1n = B2n
H2t − H1t = Ks
通俗解释

B 法向连续来自 ∇·B=0；H 切向条件来自安培环路定理。

符号拆解
  • B₁n、B₂n 是法向磁感应强度。
  • H₁t、H₂t 是切向磁场强度。
  • K_s 是传导面电流密度。
什么时候用

磁场跨介质边界、磁场折射、磁屏蔽问题使用。

易错提醒

无面电流时 H_t 连续；不要误以为所有情况下 H_t 都连续。

小题：无面电流时 H₁t 与 H₂t 关系？
  1. 边界条件：H₂t−H₁t=K_s。
  2. 无面电流：K_s=0。
  3. 因此 H₂t=H₁t。
答案整理：H₁t=H₂t。

5.6 理想导磁体：μ→∞ 时 H 趋于 0

理想导磁体极限思想

理想导磁体磁导率 μ→∞。若 B 有限，由 B=μH 可知 H 必须趋于 0。

H = Bμ，当 μ→∞ 且 B 有限时，H→0
通俗解释

这和理想导电体中 E→0 的逻辑相似：材料越容易导磁，维持有限 B 所需的 H 越小。

符号拆解
  • μ 是磁导率。
  • B 是磁感应强度。
  • H 是磁场强度。
什么时候用

理解理想磁边界、强磁导材料近似时使用。

易错提醒

H 趋于 0 不等于 B 一定为 0，二者不是同一个量。

小题：μ→∞ 且 B 有限，H 为多少？
  1. 由 B=μH 得 H=B/μ。
  2. B 有限，μ 无限大。
  3. 有限量除以无限大趋于 0。
答案整理：H→0。

5.7 磁矩与转矩：载流线圈在磁场中会转动

磁矩转矩

载流线圈相当于一个磁偶极子，放入外磁场后会受到转矩，使磁矩趋向与磁场方向一致。

m = NIS n
T = m×B
|T| = mB sinθ
通俗解释

线圈像一个小指南针，外磁场会让它转到稳定方向。θ=90° 时转矩最大，θ=0° 时稳定平衡。

符号拆解
  • m 是磁矩。
  • N 是匝数。
  • I 是电流。
  • S 是面积。
  • B 是磁感应强度。
  • T 是转矩。
什么时候用

求载流线圈受力矩、平衡位置时使用。

易错提醒

线圈平面与 B 平行时，磁矩 m 与 B 垂直，不要把线圈平面方向和法向方向混淆。

小题：N=100，I=2A，S=0.02m²，B=0.5T，θ=60°，求转矩。
  1. 先求磁矩：m=NIS=100×2×0.02=4 A·m²。
  2. 用 |T|=mBsinθ。
  3. 代入：|T|=4×0.5×sin60°。
  4. sin60°=√3/2，所以 |T|=√3≈1.73 N·m。
答案整理：|T|≈1.73 N·m。
