恒定电流场

本章把电场放进导电介质里，研究稳态电流如何流动、如何满足连续性以及如何产生功率损耗。

本章独立学习路线

  1. 电流密度 J理解电流不只是总量，还可以是空间分布。
  2. 连续性掌握 ∇·J=0 的物理意义。
  3. 边界条件学会跨介质处理 J 和 E。
  4. 静电比拟把静电场方法迁移到电流场。

本章与前后章节的联系

第四章和第二章的数学结构非常像：静电场中 D=εE，电流场中 J=σE。第五章会从电场转向磁场，研究电流如何产生磁场。

4.1 恒定电流场：电荷持续定向运动形成电流

恒定电流电流密度

恒定电流场研究稳态导电介质中的电流分布。稳态意味着各处电流不随时间改变。

J = σE
I = ∬SJ·dS
通俗解释

J 描述单位面积上通过多少电流；σE 说明导电材料中电场推动电荷形成电流。

符号拆解
  • J 是电流密度，单位 A/m²。
  • σ 是电导率。
  • E 是电场强度。
  • I 是通过截面的总电流。
什么时候用

求导体或导电介质中电流、电流密度时使用。

易错提醒

J 是面密度意义的矢量，不等同于总电流 I。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

电流密度 J 表示单位面积上通过的电流，J=σE 表示电场驱动导体中电流。

本知识点公式：J = σE I = ∬ S J·dS
2. 每个字母/符号代表什么
J电流密度，单位 A/m²。
σ电导率，单位 S/m。
E电场强度。
I总电流。
3. 遇到题目怎么算
  1. 若已知 σ 和 E，直接 J=σE。
  2. 若求总电流，用 I=∬J·dS。
  3. 注意 J 是面密度，I 是总量。
4. 常见题型
  • 由 E 求 J
  • 由 J 求 I
  • 均匀截面电流
  • 导电介质欧姆定律
速算例子：σ=2S/m，E=5V/m，则 J=10A/m²。
小题：σ=2 S/m，E=5 V/m，求 J。
  1. 使用欧姆定律微分形式 J=σE。
  2. 代入 σ=2，E=5。
  3. J=10 A/m²，方向与 E 相同。
答案整理：J=10 A/m²。

4.2 恒定电流场方程：电流连续且无旋电场

场方程连续性

稳态时没有电荷持续堆积，所以电流密度散度为零；恒定电场仍满足静电型无旋条件。

∇·J = 0
∯SJ·dS = 0
∇×E = 0
∮LE·dl = 0
通俗解释

∇·J=0 表示电流线没有起点和终点，不会在某处凭空产生或消失。

符号拆解
  • J 是电流密度。
  • E 是电场强度。
  • S 是闭合面。
  • L 是闭合曲线。
什么时候用

判断稳态电流分布和建立边界条件时使用。

易错提醒

∇·J=0 不是说 J=0，而是说流入和流出平衡。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

恒定电流连续性表示稳态时电荷不在某点堆积，因此 ∇·J=0。

本知识点公式：∇·J = 0 ∯ S J·dS = 0 ∇×E = 0 ∮ L E·dl = 0
2. 每个字母/符号代表什么
∇·J电流密度散度。
J电流密度。
0无净源汇、无电荷积累。
3. 遇到题目怎么算
  1. 判断是否恒定/稳态。
  2. 稳态直接写 ∇·J=0。
  3. 对闭合面可写 ∯J·dS=0。
4. 常见题型
  • 稳态电流判断
  • 闭合面电流通量
  • 连续性方程简化
  • 电流线连续判断
速算例子：恒定电流场中闭合面净流出为 0。
小题：闭合面内恒定电流净流出为多少？
  1. 恒定电流满足 ∯J·dS=0。
  2. 这表示任意闭合面净电流为零。
  3. 所以流入等于流出。
答案整理：净流出为 0。

4.3 电流连续性原理：电流线必须闭合

连续性物理意义

恒定电流线没有起点和终点，必须连续闭合。这和静电场线不同，静电场线可以起于正电荷、止于负电荷。

稳态：∇·J = 0
通俗解释

如果某处电流线突然终止，就意味着电荷在那里不断堆积，这就不是恒定状态。

符号拆解
  • ∇·J 表示电流密度的散度。
  • 等于 0 表示没有净源汇。
什么时候用

解释导电介质中电流分布、边界法向电流连续时使用。

易错提醒

不要把电流线和电场线混同。电流线在稳态中闭合，静电场线可起止于电荷。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

恒定电流边界条件保证界面处电荷不无限积累：法向 J 连续，切向 E 连续。

本知识点公式：稳态：∇·J = 0
2. 每个字母/符号代表什么
J1n,J2n界面两侧电流密度法向分量。
E1t,E2t界面两侧电场切向分量。
σ不同介质的电导率。
3. 遇到题目怎么算
  1. 分解为法向和切向。
  2. 法向用 J1n=J2n。
  3. 切向用 E1t=E2t。
  4. 用 J=σE 转换不同介质关系。
4. 常见题型
  • 导电介质分界面
  • 电流折射
  • 界面电荷积累判断
  • 边界条件计算
速算例子：稳态无积累时，J 的法向分量连续。
小题：为什么稳态导线中电流处处相等？
  1. 若某截面电流大、另一截面小，中间会出现电荷堆积或减少。
  2. 稳态不允许持续堆积。
  3. 所以沿串联导线路径电流连续。
答案整理：因为恒定电流满足连续性，不能在导线中途凭空产生或消失。

4.4 恒定电流场边界条件

边界条件导电介质

两种导电介质交界面上，法向电流密度连续，切向电场强度连续。

J1n = J2n
E1t = E2t
通俗解释

法向 J 连续保证电流不在界面堆积；切向 E 连续来自恒定电场的无旋性。

符号拆解
  • J₁n、J₂n 是界面两侧法向电流密度。
  • E₁t、E₂t 是界面两侧切向电场。
  • n 是法向，t 是切向。
什么时候用

分层导电介质、电流折射问题使用。

易错提醒

法向连续的是 J，不一定是 E；因为 J=σE，σ 不同时 E_n 会改变。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

电流场中的功率损耗表示单位体积内电能转化为热能的速率。

本知识点公式：J 1n = J 2n E 1t = E 2t
2. 每个字母/符号代表什么
p功率损耗密度，单位 W/m³。
J电流密度。
E电场强度。
σ电导率。
3. 遇到题目怎么算
  1. 已知 J 和 E，用 p=J·E。
  2. 已知 σ 和 E，用 p=σE²。
  3. 已知 J 和 σ，用 p=J²/σ。
  4. 总功率再对体积积分。
4. 常见题型
  • 功率密度计算
  • 总损耗计算
  • 由损耗反求 E/J
  • 导体发热题
速算例子：σ=4，E=3，则 p=σE²=36W/m³。
小题：σ₁≠σ₂ 且 J₁n=J₂n，E₁n 与 E₂n 一定相等吗？
  1. 由 J=σE 得 J_n=σE_n。
  2. 边界条件为 σ₁E₁n=σ₂E₂n。
  3. 若 σ₁≠σ₂，则 E₁n 一般不等于 E₂n。
答案整理：不一定相等。

4.5 导电介质功率损耗

功率焦耳热

电流在导电介质中流动会产生热损耗。单位体积损耗由 J·E 描述。

p = J·E = σE² = J²σ
P = ∭Vp dV
通俗解释

电场推动电荷运动，介质阻碍运动，这部分能量转化为热。

符号拆解
  • p 是单位体积功率损耗。
  • P 是总功率损耗。
  • σ 是电导率。
  • V 是导电区域体积。
什么时候用

求导体发热、介质损耗时使用。

易错提醒

注意 p 是功率密度，P 才是总功率。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

静电比拟利用静电场和恒定电流场数学结构相似，把已知电容结果转成电导结果。

本知识点公式：p = J·E = σE² = J² σ P = ∭ V p dV
2. 每个字母/符号代表什么
σ↔ε电导率对应介电常数。
J↔D电流密度对应电位移。
G↔C电导对应电容。
3. 遇到题目怎么算
  1. 确认几何结构相同。
  2. 把静电问题中的 ε 替换为 σ。
  3. 把 D 替换为 J，把 C 替换为 G。
4. 常见题型
  • 由电容求电导
  • 同轴/平行板比拟
  • 静电场解迁移
  • 材料参数替换
速算例子：若几何相同，常有 G=(σ/ε)C。
小题：σ=4 S/m，E=3 V/m，求 p。
  1. 用 p=σE²。
  2. 代入 σ=4，E=3。
  3. p=4×9=36 W/m³。
答案整理：p=36 W/m³。

4.6 静电比拟：恒定电场与静电场的相似性

比拟电导电容

均匀导电媒质中的恒定电场，与均匀介质无源区中的静电场形式相似，可以用静电问题的结果类比电流问题。

σ ↔ ε
J ↔ D
G ↔ C
G = (σ/ε)C
通俗解释

把电导率 σ 看成介电常数 ε 的对应物，把电流密度 J 看成电位移 D 的对应物，就能迁移解法。

符号拆解
  • σ 是电导率。
  • ε 是介电常数。
  • G 是电导。
  • C 是电容。
什么时候用

几何形状相同、边界条件相同时，用已知电容推电导。

易错提醒

比拟要求边界条件和几何结构对应，不能随意套。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

静电比拟利用静电场和恒定电流场数学结构相似，把电容、电位移等静电结果转成电导、电流密度等电流场结果。

本知识点公式：σ ↔ ε J ↔ D G ↔ C G = (σ/ε)C
2. 每个字母/符号代表什么
σ电导率，描述材料导电能力。
ε介电常数，描述材料对电场的响应。
J电流密度，对应静电场中的 D。
D电位移矢量，对应电流场中的 J。
G电导，对应静电问题中的电容 C。
C电容。
3. 遇到题目怎么算
  1. 确认两个问题几何结构相同。
  2. 把静电场里的 ε 替换成电流场里的 σ。
  3. 把 D 替换为 J，把 C 替换为 G。
  4. 如果题目给 C 和 σ/ε，直接用 G=(σ/ε)C。
4. 常见题型
  • 由电容求电导
  • 平行板/同轴结构比拟
  • 材料参数替换题
  • 静电解迁移到电流场
速算例子：C=2pF，σ/ε=10^6s^-1，则 G=2×10^-12×10^6=2×10^-6S。
小题：某结构 C=2 pF，σ/ε=10⁶ s⁻¹，求 G。
  1. 用 G=(σ/ε)C。
  2. C=2×10⁻¹² F。
  3. G=10⁶×2×10⁻¹²=2×10⁻⁶ S。
答案整理：G=2 μS。

4.7 理想导电体中不存在恒定电场

理想导体极限思想

理想导电体电导率 σ→∞。若有限电流密度 J 存在，由 J=σE 可知 E 必须趋于 0。

E = Jσ，当 σ→∞ 且 J 有限时，E→0
通俗解释

理想导体几乎不需要电场就能维持电流，因此内部恒定电场为零。

符号拆解
  • σ 是电导率。
  • J 是电流密度。
  • E 是电场强度。
什么时候用

判断理想导体、电磁边界理想化时使用。

易错提醒

现实导体不是 σ 无限大，实际导线内部可以有很小电场。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

理想导电体的电导率趋于无穷大。若电流密度 J 有限，由 E=J/σ 可知内部恒定电场趋于 0。

本知识点公式：E = J σ ，当 σ→∞ 且 J 有限时，E→0
2. 每个字母/符号代表什么
E导体内部电场。
J电流密度。
σ电导率。
σ→∞理想导体近似。
3. 遇到题目怎么算
  1. 从欧姆定律 J=σE 变形得到 E=J/σ。
  2. 判断 J 是否有限。
  3. 令 σ→∞，分母趋于无穷，E→0。
4. 常见题型
  • 理想导体极限分析
  • 导体内部电场判断
  • 由 J 和 σ 求 E
  • 导电率变化趋势题
速算例子：若 J 有限且 σ→∞，则 E=J/σ→0。
小题：理想导体中若 J 有限，E 为多少？
  1. 由 J=σE 得 E=J/σ。
  2. 理想导体 σ→∞。
  3. 有限 J 除以无限大趋于 0。
答案整理：E=0。
