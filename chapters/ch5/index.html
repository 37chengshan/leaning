第 5 章

恒定磁场

本章研究恒定电流如何产生磁场，以及磁场在介质中如何表现、如何跨边界变化、如何对载流线圈产生转矩。

本章独立学习路线

  1. 从电流到磁场理解恒定电流是磁场旋度的来源。
  2. B 与 H区分磁感应强度和磁场强度。
  3. 磁化理解材料内部磁矩排列造成的宏观效应。
  4. 边界和转矩掌握磁介质边界条件与线圈受力矩。

本章与前后章节的联系

第五章建立磁场的静态规律。第六章麦克斯韦方程组会把电场和磁场统一起来：变化的电场能产生磁场，变化的磁场也能产生电场。

5.1 真空中恒定磁场方程：有旋无散

磁场安培环路定理

恒定磁场由恒定电流产生。磁场没有磁单极子，所以 B 的散度为零；电流会激发磁场旋度。

∇×B = μ₀J
∇·B = 0
∮LB·dl = μ₀I
通俗解释

电流像磁场的旋涡源；磁感应线没有头尾，总是闭合的。

符号拆解
  • B 是磁感应强度。
  • J 是电流密度。
  • μ₀ 是真空磁导率。
  • I 是闭合曲线包围的电流。
什么时候用

分析真空中电流产生磁场时使用。

易错提醒

磁场线没有起点和终点，不像静电场线起于正电荷止于负电荷。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

恒定磁场由恒定电流产生，H 的旋度与电流密度相关，B 无散表示磁力线闭合。

本知识点公式：∇×B = μ₀J ∇·B = 0 ∮ L B·dl = μ₀I
2. 每个字母/符号代表什么
H磁场强度，单位 A/m。
B磁感应强度，单位 T。
J电流密度。
I闭合路径所包围电流。
3. 遇到题目怎么算
  1. 有对称电流时用安培环路定理 ∮H·dl=I。
  2. 磁场线闭合对应 ∇·B=0。
  3. 根据几何选合适闭合路径。
4. 常见题型
  • 安培环路定理求 H
  • 长直导线磁场
  • 环形磁路
  • 判断 B 无散
速算例子：闭合路径包围 5A 电流，则 ∮H·dl=5A。
小题：为什么 ∇·B=0 表示磁力线闭合？
  1. 散度表示源汇。
  2. ∇·B=0 表示磁场没有源或汇。
  3. 没有源汇的场线不能凭空开始或结束，只能闭合或延伸到无穷远。
答案整理：表示磁场无源，磁力线无头无尾。

5.2 介质中的恒定磁场方程：用 H 处理传导电流

H介质磁场

在磁介质中，用磁场强度 H 表述安培环路定理，右边只考虑传导电流。

∇×H = J
∇·B = 0
∮LH·dl = I
通俗解释

H 的引入类似 D：把介质磁化影响从源项中分离出来，让方程右边主要看传导电流。

符号拆解
  • H 是磁场强度，单位 A/m。
  • B 是磁感应强度，单位 T。
  • J 是传导电流密度。
  • I 是传导电流。
什么时候用

磁介质中使用安培环路定理时常用 H。

易错提醒

不要把 B 和 H 混为一谈，它们通过介质本构关系联系。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

矢量磁位 A 用于表示 B，因为 ∇·B=0 时可令 B=∇×A。

本知识点公式：∇×H = J ∇·B = 0 ∮ L H·dl = I
2. 每个字母/符号代表什么
A矢量磁位。
B磁感应强度。
∇×AA 的旋度。
3. 遇到题目怎么算
  1. 若题目要求用磁位描述磁场，写 B=∇×A。
  2. 再根据规范条件或边界条件求 A。
  3. 最后由旋度求 B。
4. 常见题型
  • 磁位概念题
  • 由 A 求 B
  • 判断矢量势存在性
  • 磁场方程推导题
速算例子：只要 B=∇×A，则自动满足 ∇·B=0。
小题：若闭合路径包围电流 5 A，则 ∮H·dl 等于多少？
  1. 介质中安培环路定理：∮H·dl=I。
  2. 题中 I=5 A。
  3. 直接得到结果。
答案整理：∮H·dl=5 A。

5.3 磁介质本构关系：B、H、M 的联系

磁化本构关系

磁介质在外磁场作用下会磁化。磁化强度 M 描述单位体积内磁矩的合成效果。

B = μ₀(H + M)
线性各向同性介质：B = μH = μrμ₀H
通俗解释

H 描述由传导电流激发的磁场强度，M 描述材料被磁化后的贡献，B 是综合后的磁感应强度。

符号拆解
  • M 是磁化强度。
  • μ 是磁导率。
  • μᵣ 是相对磁导率。
  • μ₀ 是真空磁导率。
什么时候用

处理磁介质、铁磁材料、磁屏蔽时使用。

易错提醒

线性关系 B=μH 不适用于所有强非线性铁磁材料的全过程。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

磁介质中 B、H、M 描述不同层次：H 与传导电流关系密切，M 描述材料磁化，B 是总磁感应强度。

本知识点公式：B = μ₀(H + M) 线性各向同性介质：B = μH = μ r μ₀H
2. 每个字母/符号代表什么
B磁感应强度，单位 T。
H磁场强度，单位 A/m。
M磁化强度。
μ, μr, μ0磁导率、相对磁导率、真空磁导率。
3. 遇到题目怎么算
  1. 线性各向同性介质中直接用 B=μH。
  2. 若给 μr，则 μ=μrμ0。
  3. 已知 H 后代入求 B。
4. 常见题型
  • 由 H 求 B
  • 求相对磁导率
  • 磁介质本构关系
  • B/H/M 区分
速算例子：μr=100，H=10A/m，B=100μ0×10≈1.26mT。
小题：μᵣ=100，H=10 A/m，求 B 大小。
  1. 用 B=μᵣμ₀H。
  2. 代入 μᵣ=100，H=10。
  3. B=1000μ₀ T。
  4. 若 μ₀=4π×10⁻⁷，则 B≈1.26×10⁻³ T。
答案整理：B=1000μ₀ T≈1.26 mT。

5.4 磁化电流：磁化也等效为电流

磁化电流M

材料内部微观电流环在宏观上产生磁化效果，可以等效为磁化电流体密度和面密度。

Jm = ∇×M
Km = M×n
通俗解释

如果 M 在空间中有旋转变化，会形成体磁化电流；如果磁化介质有表面，会出现面磁化电流。

符号拆解
  • J_m 是磁化电流体密度。
  • K_m 是磁化电流面密度。
  • M 是磁化强度。
  • n 是表面法向单位矢量。
什么时候用

分析磁化介质等效电流时使用。

易错提醒

磁化电流不是外接电源提供的传导电流，而是材料内部磁化的等效描述。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

磁化也可以等效为电流：体磁化电流由 ∇×M 给出，面磁化电流由 M×n 给出。

本知识点公式：J m = ∇×M K m = M×n
2. 每个字母/符号代表什么
Jm体磁化电流密度。
Km面磁化电流密度。
M磁化强度。
n边界外法向单位矢量。
3. 遇到题目怎么算
  1. 判断 M 是否随空间变化。
  2. 若求体磁化电流，算 ∇×M。
  3. 若求表面磁化电流，算 M×n。
4. 常见题型
  • 体磁化电流
  • 面磁化电流
  • 均匀磁化判断
  • 右手定则方向题
速算例子：均匀 M 无空间变化时，∇×M=0，所以 Jm=0。
小题：均匀 M 且无空间变化时，J_m 如何？
  1. J_m=∇×M。
  2. 若 M 均匀不随空间变化，各偏导为 0。
  3. 因此旋度为 0。
答案整理：J_m=0，但表面仍可能有 K_m。

5.5 恒定磁场边界条件

边界条件磁介质

两种磁介质分界面上，B 的法向分量连续；H 的切向分量是否跳变取决于界面传导面电流。

B1n = B2n
H2t − H1t = Ks
通俗解释

B 法向连续来自 ∇·B=0；H 切向条件来自安培环路定理。

符号拆解
  • B₁n、B₂n 是法向磁感应强度。
  • H₁t、H₂t 是切向磁场强度。
  • K_s 是传导面电流密度。
什么时候用

磁场跨介质边界、磁场折射、磁屏蔽问题使用。

易错提醒

无面电流时 H_t 连续；不要误以为所有情况下 H_t 都连续。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

磁边界条件：B 的法向分量连续，H 的切向分量跳变由面电流决定。

本知识点公式：B 1n = B 2n H 2t − H 1t = K s
2. 每个字母/符号代表什么
B1n,B2n界面两侧 B 的法向分量。
H1t,H2t界面两侧 H 的切向分量。
Ks自由面电流密度。
3. 遇到题目怎么算
  1. 先画界面法向。
  2. 法向用 B1n=B2n。
  3. 切向用 H2t-H1t=Ks。
  4. 若 Ks=0，则 Ht 连续。
4. 常见题型
  • 磁介质边界
  • 有/无面电流判断
  • B/H 分量转换
  • 磁场折射关系
速算例子：无面电流时 Ks=0，因此 H2t=H1t。
小题：无面电流时 H₁t 与 H₂t 关系？
  1. 边界条件：H₂t−H₁t=K_s。
  2. 无面电流：K_s=0。
  3. 因此 H₂t=H₁t。
答案整理：H₁t=H₂t。

5.6 理想导磁体：μ→∞ 时 H 趋于 0

理想导磁体极限思想

理想导磁体磁导率 μ→∞。若 B 有限，由 B=μH 可知 H 必须趋于 0。

H = Bμ，当 μ→∞ 且 B 有限时，H→0
通俗解释

这和理想导电体中 E→0 的逻辑相似：材料越容易导磁，维持有限 B 所需的 H 越小。

符号拆解
  • μ 是磁导率。
  • B 是磁感应强度。
  • H 是磁场强度。
什么时候用

理解理想磁边界、强磁导材料近似时使用。

易错提醒

H 趋于 0 不等于 B 一定为 0，二者不是同一个量。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

理想导磁体 μ→∞ 时，在 B 有限条件下 H=B/μ 趋近于 0。

本知识点公式：H = B μ ，当 μ→∞ 且 B 有限时，H→0
2. 每个字母/符号代表什么
μ磁导率。
B磁感应强度。
H磁场强度。
3. 遇到题目怎么算
  1. 写出 H=B/μ。
  2. 判断 B 是否有限。
  3. 令 μ→∞，则 H→0。
4. 常见题型
  • 理想导磁体近似
  • 磁路边界判断
  • 极限分析
  • H 与 B 关系题
速算例子：μ→∞ 且 B 有限，则 H≈0。
小题：μ→∞ 且 B 有限，H 为多少？
  1. 由 B=μH 得 H=B/μ。
  2. B 有限，μ 无限大。
  3. 有限量除以无限大趋于 0。
答案整理：H→0。

5.7 磁矩与转矩：载流线圈在磁场中会转动

磁矩转矩

载流线圈相当于一个磁偶极子，放入外磁场后会受到转矩，使磁矩趋向与磁场方向一致。

m = NIS n
T = m×B
|T| = mB sinθ
通俗解释

线圈像一个小指南针，外磁场会让它转到稳定方向。θ=90° 时转矩最大，θ=0° 时稳定平衡。

符号拆解
  • m 是磁矩。
  • N 是匝数。
  • I 是电流。
  • S 是面积。
  • B 是磁感应强度。
  • T 是转矩。
什么时候用

求载流线圈受力矩、平衡位置时使用。

易错提醒

线圈平面与 B 平行时，磁矩 m 与 B 垂直，不要把线圈平面方向和法向方向混淆。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

磁矩描述载流线圈的磁效应，转矩表示线圈在磁场中受到的旋转作用。

本知识点公式：m = NIS n T = m×B |T| = mB sinθ
2. 每个字母/符号代表什么
m磁矩。
N线圈匝数。
I电流。
S线圈面积。
n面积法向单位矢量。
T转矩。
θm 与 B 的夹角。
3. 遇到题目怎么算
  1. 先算磁矩大小 m=NIS。
  2. 再算转矩大小 |T|=mBsinθ。
  3. 方向用右手定则判断 m×B。
4. 常见题型
  • 线圈磁矩
  • 转矩大小计算
  • 最大/最小转矩
  • 方向判断
速算例子：N=100,I=2,S=0.02,B=0.5,θ=60°，T=100×2×0.02×0.5×sin60°≈1.73N·m。
小题：N=100，I=2A，S=0.02m²，B=0.5T，θ=60°，求转矩。
  1. 先求磁矩：m=NIS=100×2×0.02=4 A·m²。
  2. 用 |T|=mBsinθ。
  3. 代入：|T|=4×0.5×sin60°。
  4. sin60°=√3/2，所以 |T|=√3≈1.73 N·m。
答案整理：|T|≈1.73 N·m。
