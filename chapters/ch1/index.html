矢量分析基础

本章是后面所有电场、磁场公式的数学语言。先理解场、矢量、∇、梯度、散度、旋度，后面才不会只是在背公式。

本章独立学习路线

  1. 场与矢量知道物理量如何分布在空间里。
  2. ∇ 算子理解空间变化如何被数学表达。
  3. 梯度/散度/旋度分别理解变化最快、源汇、旋转。
  4. 积分定理把局部性质和整体性质联系起来。

1.1 场：空间中每一点都有一个物理量

概念入门

场不是一个单独的数，而是一整片空间里的分布规律。比如教室里每个位置都有温度，这就是温度场；空间中每一点都有电场强度，这就是电场。

标量场：T = T(x, y, z, t)
矢量场：A = A(x, y, z, t)
通俗解释

你可以把场想成一张“空间地图”：地图上每个位置都写着一个数，或者画着一个箭头。

符号拆解
  • T 表示温度这类只有大小的量。
  • A 表示矢量场，既有大小又有方向。
  • x、y、z 表示空间位置。
  • t 表示时间。
什么时候用

看到题目说“某区域内每一点都有某个物理量”时，就要想到场。

易错提醒

不要把场理解成某一个点的量。某一点的值只是场在该点的取值。

小题：判断温度、电场、速度哪个属于场？
  1. 温度如果随空间位置变化，可写成 T(x,y,z)，是标量场。
  2. 电场强度 E(x,y,z) 在每点有大小和方向，是矢量场。
  3. 流体速度 v(x,y,z) 在每点有速度大小和方向，也是矢量场。
答案整理：三者都可以构成场；温度是标量场，电场和速度是矢量场。

1.2 标量与矢量：只有大小，还是既有大小又有方向

基础坐标分量

标量只有大小，比如温度、电位；矢量既有大小又有方向，比如力、速度、电场强度、磁场强度。

A = Axex + Ayey + Azez
|A| = √(Ax2 + Ay2 + Az2)
通俗解释

一个空间箭头可以拆成三个方向上的分量。模长就是这个箭头的真实长度，本质上是三维勾股定理。

符号拆解
  • A 是矢量。
  • Aₓ、Aᵧ、A_z 是三个坐标方向的分量。
  • eₓ、eᵧ、e_z 是三个单位矢量。
  • |A| 是矢量大小。
什么时候用

给出矢量分量时，用这个式子表达矢量或求矢量大小。

易错提醒

不要把分量当成矢量本身。分量只是数，乘上单位矢量后才说明方向。

小题：A=3eₓ+4eᵧ，求 |A|。
  1. 识别分量：Aₓ=3，Aᵧ=4，A_z=0。
  2. 代入模长公式：|A|=√(3²+4²+0²)。
  3. 计算：√25=5。
答案整理：|A|=5。

1.3 点积与叉积：矢量相乘的两种方式

运算方向

点积衡量两个矢量同向程度，结果是标量；叉积衡量两个矢量构成的垂直效果，结果是矢量。

A·B = |A||B|cosθ = AxBx + AyBy + AzBz
|A×B| = |A||B|sinθ
通俗解释

点积像把一个矢量投影到另一个方向上；叉积像用右手定则判断旋转轴方向。

符号拆解
  • θ 是 A 和 B 的夹角。
  • cosθ 描述同向程度。
  • sinθ 描述垂直程度。
  • A×B 的方向由右手定则决定。
什么时候用

算功、通量、环量常用点积；算力矩、旋度方向、磁场力常用叉积。

易错提醒

点积结果不是矢量；叉积顺序不能随便换，A×B = −B×A。

小题：A=2eₓ+eᵧ，B=eₓ+3eᵧ，求 A·B。
  1. 写出分量：Aₓ=2，Aᵧ=1；Bₓ=1，Bᵧ=3。
  2. 用点积分量公式：A·B=AₓBₓ+AᵧBᵧ。
  3. 代入：2×1+1×3=5。
答案整理：A·B=5。

1.4 ∇ 算子：空间变化检测器

核心工具偏导

∇ 不是普通数字，而是由三个方向偏导数组成的微分算子。它负责检查一个场沿 x、y、z 三个方向怎么变化。

∇ = ex∂∂x + ey∂∂y + ez∂∂z
通俗解释

它像一个三方向扫描器：分别扫描 x、y、z 方向上的变化。后面接标量得到梯度，点乘矢量得到散度，叉乘矢量得到旋度。

符号拆解
  • ∂/∂x 表示只看 x 方向变化。
  • ∂/∂y 表示只看 y 方向变化。
  • ∂/∂z 表示只看 z 方向变化。
  • eₓ、eᵧ、e_z 标明方向。
什么时候用

遇到梯度、散度、旋度时，都先想到 ∇。

易错提醒

∇ 本身不是结果，它必须作用到某个标量场或矢量场上才有意义。

小题：∇φ、∇·A、∇×A 分别叫什么？
  1. ∇ 作用于标量 φ，得到 ∇φ，叫梯度。
  2. ∇ 与矢量 A 做点积，得到 ∇·A，叫散度。
  3. ∇ 与矢量 A 做叉积，得到 ∇×A，叫旋度。
答案整理：∇φ 是梯度，∇·A 是散度，∇×A 是旋度。

1.5 梯度与方向导数：标量场变化最快的方向

梯度方向导数

梯度用于标量场，告诉你该标量在哪个方向增加最快，以及增加得有多快。方向导数则是问沿某个指定方向变化多快。

∇φ = ex∂φ∂x + ey∂φ∂y + ez∂φ∂z
∂φ∂l = ∇φ · el
通俗解释

梯度像山坡上最陡的上坡方向。方向导数像你不一定沿最陡方向走，而是沿题目指定方向走，看看高度变化多快。

符号拆解
  • φ 是标量场。
  • ∇φ 是梯度矢量。
  • e_l 是指定方向的单位矢量。
  • ∂φ/∂l 是沿 l 方向变化率。
什么时候用

求温度、电位、高度等标量场变化方向时用梯度；题目问“沿某方向变化率”时用方向导数。

易错提醒

方向向量必须先单位化，否则方向导数会被长度放大。

小题：φ=x²+y²，求点(1,2)处梯度。
  1. 求偏导：∂φ/∂x=2x，∂φ/∂y=2y。
  2. 代入点(1,2)：∂φ/∂x=2，∂φ/∂y=4。
  3. 写成矢量：∇φ=2eₓ+4eᵧ。
答案整理：∇φ|(1,2)=2eₓ+4eᵧ。

1.6 散度与通量：有没有源、有没有净流出

散度通量

散度是局部概念，判断某点附近是否像源一样向外发散；通量是整体概念，计算穿过一个面的总量。

∇·A = ∂Ax∂x + ∂Ay∂y + ∂Az∂z
Φ = ∬S A·dS
通俗解释

散度像看一个点是不是喷泉或排水口。通量像看一张网有多少水流穿过去。

符号拆解
  • A 是矢量场。
  • Aₓ、Aᵧ、A_z 是分量。
  • S 是曲面。
  • dS 是带方向的小面积。
什么时候用

判断场是否有源时用散度；求穿过某个面或闭合面的总量时用通量。

易错提醒

通量为零不一定说明没有场，可能是进出抵消。

小题：A=xeₓ+yeᵧ+ze_z，求散度。
  1. 识别分量：Aₓ=x，Aᵧ=y，A_z=z。
  2. 逐项求偏导：∂Aₓ/∂x=1，∂Aᵧ/∂y=1，∂A_z/∂z=1。
  3. 相加：∇·A=3。
答案整理：散度为 3，表示该场处处有向外发散趋势。

1.7 旋度与环量：有没有局部打转

旋度环量

旋度判断矢量场局部有没有旋转趋势；环量计算沿闭合路径走一圈的总绕行效果。

∇×A = (∂Az∂y − ∂Ay∂z)ex + (∂Ax∂z − ∂Az∂x)ey + (∂Ay∂x − ∂Ax∂y)ez
Γ = ∮LA·dl
通俗解释

把小叶轮放进场里，如果叶轮会转，说明旋度不为零。环量则是沿一整条闭合曲线累加这个绕行效果。

符号拆解
  • ∇×A 是旋度，结果是矢量。
  • Γ 是环量。
  • L 是闭合曲线。
  • dl 是沿路径方向的小线段。
什么时候用

判断场是否无旋、是否能写成某个势函数梯度时用旋度。

易错提醒

旋度公式正负号容易写反，建议按行列式顺序记。

小题：A=−yeₓ+xeᵧ，求旋度 z 分量。
  1. 识别 Aₓ=−y，Aᵧ=x。
  2. z 分量公式：(∇×A)_z=∂Aᵧ/∂x−∂Aₓ/∂y。
  3. 计算：∂x/∂x=1，∂(−y)/∂y=−1。
  4. 相减：1−(−1)=2。
答案整理：(∇×A)_z=2，说明绕 z 轴有旋转趋势。

1.8 散度定理与斯托克斯定理：局部和整体互相转换

积分定理高斯定理

散度定理把体内散度与边界通量联系起来；斯托克斯定理把曲面上的旋度与边界环量联系起来。

∯SA·dS = ∭V(∇·A)dV
∮LA·dl = ∬S(∇×A)·dS
通俗解释

第一条说：内部所有源的总和等于边界总流出。第二条说：边界绕行效果等于内部所有局部旋转效果的总和。

符号拆解
  • S 是闭合曲面。
  • V 是曲面包围的体积。
  • L 是曲面的边界曲线。
  • dV 是体积元。
什么时候用

复杂面积分和体积分之间转换时使用；后面高斯定律、安培环路定理都会用到这些思想。

易错提醒

散度定理要求 S 是闭合曲面；斯托克斯定理的 L 是曲面 S 的边界。

小题：闭合面通量为正说明什么？
  1. 通量是穿过闭合面的净流出量。
  2. 通量为正，说明流出多于流入。
  3. 用散度定理看，内部散度积分为正，表示内部存在净源。
答案整理：闭合面内存在净源，整体向外流出。
