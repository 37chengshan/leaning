静电场

本章把第一章的数学工具用到电荷和电场中，核心是 E、φ、D、边界条件和静电能量。

  1. 电场强度先理解电场是单位正电荷受到的力。
  2. 电位再用标量 φ 简化静电场计算。
  3. 介质与导体理解材料对电场的影响。
  4. 边界与能量掌握分界面和储能问题。

本章与前后章节的联系

第二章直接使用第一章的散度、旋度和梯度。第三章会进一步把电位 φ 用于边值问题，例如泊松方程、拉普拉斯方程和镜像法。

2.1 电场强度 E：单位正电荷受到的力

静电场定义

电场强度描述某点电场有多强、方向朝哪里。它的方向规定为正电荷在该点受到的力的方向。

E = Fq
通俗解释

把一个很小的正试探电荷放到某点，看它受到多少力。每 1 C 电荷受到的力，就是该点电场强度。

符号拆解
  • E 是电场强度，单位 N/C 或 V/m。
  • F 是电场力。
  • q 是试探电荷量。
什么时候用

题目给电场力和试探电荷时，用定义式。

易错提醒

如果试探电荷是负电荷，受力方向和电场方向相反。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

电场强度 E 表示单位正试探电荷在某点受到的电场力。

本知识点公式：E = F q
2. 每个字母/符号代表什么
E电场强度，单位 N/C 或 V/m。
F电场力，单位 N。
q试探电荷量，单位 C。
3. 遇到题目怎么算
  1. 把 F、q 化为国际单位。
  2. 代入 E=F/q。
  3. 若 q 为正，E 与 F 同向；若 q 为负，E 与 F 反向。
4. 常见题型
  • 已知力和电荷求 E
  • 已知 E 和 q 求 F
  • 判断电场方向
  • 单位换算题
速算例子：F=6mN，q=2μC，则 E=3×10³N/C。
小题：q=2 μC，受力 6 mN，求 E。
  1. 统一单位：q=2×10⁻⁶ C，F=6×10⁻³ N。
  2. 代入 E=F/q。
  3. E=(6×10⁻³)/(2×10⁻⁶)=3×10³ N/C。
  4. 若 q 为正，E 方向与 F 相同。
答案整理：E=3×10³ N/C，方向与正电荷受力方向一致。

2.2 电场线：用图像看方向和强弱

电场线直观理解

电场线是人为画出的辅助线。切线方向表示电场方向，疏密表示电场强弱。

dxEx = dyEy = dzEz
通俗解释

这表示电场线在任一点的前进方向，都和该点电场矢量方向一致。

符号拆解
  • dx、dy、dz 是电场线小位移分量。
  • Eₓ、Eᵧ、E_z 是电场分量。
  • 比例相等说明线的切向与 E 平行。
什么时候用

判断电场线方向、画电场分布示意图时使用。

易错提醒

电场线不是实际存在的绳子，它只是辅助表示。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

电场线是辅助图像，切线方向表示 E 方向，疏密表示 E 强弱。

本知识点公式：dx E x = dy E y = dz E z
2. 每个字母/符号代表什么
dx,dy,dz电场线小位移分量。
Ex,Ey,Ez电场强度在三个方向上的分量。
3. 遇到题目怎么算
  1. 电场线方向看切线。
  2. 电场强弱看疏密。
  3. 用 dx/Ex=dy/Ey=dz/Ez 表示电场线方向与 E 平行。
4. 常见题型
  • 画电场线
  • 判断电场强弱
  • 由电场分量写电场线方程
  • 电场线性质判断
速算例子：电场线越密，表示该区域 E 越大。
小题：电场线越密代表什么？
  1. 电场线疏密用来表示电场强弱。
  2. 越密说明单位面积穿过的电场线越多。
  3. 因此该区域电场强度较大。
答案整理：电场线越密，电场越强。

2.3 电位 φ 与电场 E：电场指向电位降低最快方向

电位负梯度

静电场中可以用标量电位 φ 描述电场。电场强度等于电位梯度的负值。

E = −∇φ
φA − φB = ∫ABE·dl
通俗解释

电位像高度，正电荷会沿电位降低方向运动，所以电场方向是电位下降最快方向。

符号拆解
  • φ 是电位，单位 V。
  • ∇φ 指向电位增加最快方向。
  • 负号说明 E 指向电位降低方向。
  • E·dl 表示沿路径方向的电场贡献。
什么时候用

已知电位求电场时，用 E=−∇φ；求两点电位差时用线积分。

易错提醒

最常见错误是漏掉负号。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

电位 φ 是标量，电场 E 是电位下降最快方向，数学上 E=-∇φ。

本知识点公式：E = −∇φ φ A − φ B = ∫ A B E·dl
2. 每个字母/符号代表什么
φ电位，单位 V。
E电场强度。
∇φ电位增加最快方向。
dl路径小位移。
3. 遇到题目怎么算
  1. 已知 φ，先求梯度 ∇φ。
  2. 加负号得到 E=-∇φ。
  3. 求电位差时沿路径积分 E·dl。
4. 常见题型
  • 由电位求电场
  • 由电场求电位差
  • 判断电场方向
  • 电位等势面题
速算例子：φ=3x，则 ∇φ=3ex，E=-3ex。
小题：φ=3x，求 E。
  1. 求梯度：∇φ=3eₓ。
  2. 用 E=−∇φ。
  3. 得到 E=−3eₓ。
答案整理：E=−3eₓ V/m。

2.4 真空中静电场方程：有源无旋

高斯定律无旋

静电场有两个基本性质：电荷是电场的源，所以散度与电荷密度有关；静电场无旋，所以可以引入电位。

∇·E = ρε₀
∇×E = 0
∯SE·dS = Qε₀
∮LE·dl = 0
通俗解释

∇·E 表示电场线发散程度；∇×E=0 表示静电场没有旋涡，沿闭合路径一圈做功为零。

符号拆解
  • ρ 是体电荷密度。
  • ε₀ 是真空介电常数。
  • Q 是闭合面内总电荷。
  • S 是闭合曲面，L 是闭合路径。
什么时候用

高斯定律适合对称电荷分布求 E；无旋性质用于引入电位。

易错提醒

高斯定律总成立，但不一定总方便求解。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

真空静电场有源无旋：电荷是电场源，静电场没有旋涡。

本知识点公式：∇·E = ρ ε₀ ∇×E = 0 ∯ S E·dS = Q ε₀ ∮ L E·dl = 0
2. 每个字母/符号代表什么
ρ体电荷密度。
ε0真空介电常数。
Q闭合面内总电荷。
S/L闭合曲面/闭合路径。
3. 遇到题目怎么算
  1. 有对称电荷分布时，用高斯定律求 E。
  2. 判断静电场是否可引入电位，看 ∇×E=0。
  3. 闭合路径电场环量在静电场中为 0。
4. 常见题型
  • 高斯定律求场
  • 判断有源/无旋
  • 闭合路径积分
  • 电荷密度与散度关系题
速算例子：ρ>0 时，∇·E=ρ/ε0>0，电场线向外发散。
小题：某处 ρ>0，∇·E 是正还是负？
  1. 根据 ∇·E=ρ/ε₀。
  2. ε₀ 为正。
  3. ρ>0，所以 ∇·E>0。
答案整理：∇·E 为正，说明该处电场线向外发散。

2.5 电介质中的 D：把自由电荷单独拿出来处理

电介质电位移

介质会被电场极化。引入 D 的目的，是在介质中更方便地处理自由电荷。

D = εE = εrε₀E
∇·D = ρf
∯SD·dS = Qf
通俗解释

E 描述电场强度；D 是为了处理介质中自由电荷而引入的辅助矢量。

符号拆解
  • D 是电位移矢量，单位 C/m²。
  • ε 是介电常数。
  • εᵣ 是相对介电常数。
  • ρ_f 是自由电荷密度。
什么时候用

介质中的高斯定律通常先求 D，再用 D=εE 求 E。

易错提醒

不要把自由电荷和束缚电荷混为一谈。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

D 把介质极化影响打包，使高斯定律右边只看自由电荷。

本知识点公式：D = εE = ε r ε₀E ∇·D = ρ f ∯ S D·dS = Q f
2. 每个字母/符号代表什么
D电位移矢量，单位 C/m²。
ε介质介电常数。
εr相对介电常数。
ρf/Qf自由电荷密度/自由电荷量。
3. 遇到题目怎么算
  1. 介质问题中先看自由电荷，用 ∯D·dS=Qf 求 D。
  2. 再用 D=εE 求 E。
  3. 若 ε=εrε0，先求介质参数。
4. 常见题型
  • 介质中高斯定律
  • 由 D 求 E
  • 求相对介电常数
  • 自由电荷/束缚电荷辨析
速算例子：D=4ε0E，与 D=εrε0E 对比得 εr=4。
小题：线性介质中 D=4ε₀E，求 εᵣ。
  1. 线性介质关系：D=εᵣε₀E。
  2. 与题中 D=4ε₀E 比较。
  3. 得到 εᵣ=4。
答案整理：εᵣ=4。

2.6 导体静电平衡：内部 E=0，导体是等位体

导体等位体

导体中自由电荷能移动。静电平衡时如果内部仍有电场，电荷会继续移动，所以平衡条件是导体内部电场为零。

Einside = 0
φ = constant
通俗解释

E=−∇φ。若 E=0，则 ∇φ=0，说明导体内部电位不随位置变化，所以导体是等位体。

符号拆解
  • E_inside 是导体内部电场。
  • φ=constant 表示电位处处相同。
什么时候用

判断导体静电平衡、导体表面电荷分布时使用。

易错提醒

导体内部 E=0 不代表导体表面没有电荷，多余电荷通常在表面。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

静电平衡导体内部 E=0，因此导体内部和表面都是等位体。

本知识点公式：E inside = 0 φ = constant
2. 每个字母/符号代表什么
Einside导体内部电场。
φ电位。
constant常数，表示电位处处相同。
3. 遇到题目怎么算
  1. 静电平衡先写导体内部 E=0。
  2. 由 E=-∇φ 得 ∇φ=0。
  3. 所以 φ 不随位置变化，导体为等位体。
4. 常见题型
  • 导体静电平衡判断
  • 导体内部电场/电位
  • 导体表面电荷分布
  • 接地导体问题
速算例子：导体内部任意两点电位差为 0。
小题：静电平衡导体内部两点电位差是多少？
  1. 导体内部 E=0。
  2. 电位差与电场线积分有关。
  3. E=0，所以积分为 0。
答案整理：导体内部任意两点电位差为 0。

2.7 静电场边界条件：切向看 E，法向看 D

边界条件介质界面

场穿过不同介质分界面时，要分解成切向和法向。静电场中切向 E 连续，法向 D 的跳变由自由面电荷决定。

E1t = E2t
D2n − D1n = ρs
导体表面：Et=0，Dn=ρs
通俗解释

切向方向沿界面，法向方向垂直界面。边界条件就是告诉你场跨过界面时哪些分量不能乱跳。

符号拆解
  • t 表示切向。
  • n 表示法向。
  • ρ_s 是自由面电荷密度。
  • D=εE 可把 D 条件转成 E 条件。
什么时候用

介质分层、电场折射、导体表面电场题都要用。

易错提醒

不要把 E_t 连续误写成 D_t 连续；也不要把 D_n 条件直接当 E_n 连续。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

静电边界条件要分切向和法向：切向 E 连续，法向 D 的跳变由自由面电荷决定。

本知识点公式：E 1t = E 2t D 2n − D 1n = ρ s 导体表面：E t =0，D n =ρ s
2. 每个字母/符号代表什么
Et电场切向分量。
Dn电位移法向分量。
ρs自由面电荷密度。
1/2界面两侧介质编号。
3. 遇到题目怎么算
  1. 先画界面法向 n 和切向 t。
  2. 切向用 E1t=E2t。
  3. 法向用 D2n-D1n=ρs。
  4. 必要时用 D=εE 转换。
4. 常见题型
  • 介质界面边界条件
  • 导体表面电场
  • 求折射关系
  • 无自由面电荷判断
速算例子：ρs=0 时，D2n=D1n。
小题：无自由面电荷时，D₁n 与 D₂n 关系？
  1. 边界条件：D₂n−D₁n=ρ_s。
  2. 无自由面电荷：ρ_s=0。
  3. 所以 D₂n−D₁n=0。
答案整理：D₁n = D₂n。

2.8 静电场能量：电场本身储存能量

能量电容

电场可以对电荷做功，所以电场本身具有能量。能量密度描述单位体积储存多少能量。

we = 12E·D
We = 12∭VE·D dV
W = 12CU2 = 12QU = Q22C
通俗解释

能量密度是局部量，总能量是把所有小体积里的能量加起来。电容器公式是常用特例。

符号拆解
  • w_e 是能量密度。
  • W_e 是总能量。
  • C 是电容。
  • U 是电压，Q 是电荷量。
什么时候用

计算电场储能、电容器能量时使用。

易错提醒

公式选择看题目给什么量：给 C/U 用 1/2CU²，给 Q/C 用 Q²/2C。

深度扩展 公式含义 · 字母拆解 · 计算方法 · 常见题型
1. 这个公式/概念到底在说什么

电场能量密度表示单位体积内储存的电场能，总能量是对体积积分。

本知识点公式：w e = 1 2 E·D W e = 1 2 ∭ V E·D dV W = 1 2 CU 2 = 1 2 QU = Q 2 2C
2. 每个字母/符号代表什么
we电场能量密度。
We/W总能量。
C电容。
U/Q电压/电荷量。
3. 遇到题目怎么算
  1. 场分布已知时用 we=1/2 E·D 再积分。
  2. 电容器题看已知量选择 1/2CU²、1/2QU 或 Q²/2C。
  3. 注意单位 F、V、C、J。
4. 常见题型
  • 电场能量密度
  • 电容器储能
  • 由能量求电容/电压/电荷
  • 介质能量题
速算例子：C=2μF，U=10V，则 W=1/2CU²=1×10^-4J。
小题：C=2 μF，U=10 V，求电容器能量。
  1. 统一单位：C=2×10⁻⁶ F。
  2. 选公式 W=1/2CU²。
  3. 代入：W=1/2×2×10⁻⁶×10²。
  4. 计算：W=1×10⁻⁴ J。
答案整理：W=1.0×10⁻⁴ J。
