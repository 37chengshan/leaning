2.1 电场强度 E：单位正电荷受到的力

静电场定义

电场强度描述某点电场有多强、方向朝哪里。它的方向规定为正电荷在该点受到的力的方向。

E = Fq
通俗解释

把一个很小的正试探电荷放到某点，看它受到多少力。每 1 C 电荷受到的力，就是该点电场强度。

符号拆解
  • E 是电场强度，单位 N/C 或 V/m。
  • F 是电场力。
  • q 是试探电荷量。
什么时候用

题目给电场力和试探电荷时，用定义式。

易错提醒

如果试探电荷是负电荷，受力方向和电场方向相反。

小题：q=2 μC，受力 6 mN，求 E。
  1. 统一单位：q=2×10⁻⁶ C，F=6×10⁻³ N。
  2. 代入 E=F/q。
  3. E=(6×10⁻³)/(2×10⁻⁶)=3×10³ N/C。
  4. 若 q 为正，E 方向与 F 相同。
答案整理：E=3×10³ N/C，方向与正电荷受力方向一致。

2.2 电场线：用图像看方向和强弱

电场线直观理解

电场线是人为画出的辅助线。切线方向表示电场方向，疏密表示电场强弱。

dxEx = dyEy = dzEz
通俗解释

这表示电场线在任一点的前进方向，都和该点电场矢量方向一致。

符号拆解
  • dx、dy、dz 是电场线小位移分量。
  • Eₓ、Eᵧ、E_z 是电场分量。
  • 比例相等说明线的切向与 E 平行。
什么时候用

判断电场线方向、画电场分布示意图时使用。

易错提醒

电场线不是实际存在的绳子，它只是辅助表示。

小题：电场线越密代表什么？
  1. 电场线疏密用来表示电场强弱。
  2. 越密说明单位面积穿过的电场线越多。
  3. 因此该区域电场强度较大。
答案整理：电场线越密，电场越强。

2.3 电位 φ 与电场 E：电场指向电位降低最快方向

电位负梯度

静电场中可以用标量电位 φ 描述电场。电场强度等于电位梯度的负值。

E = −∇φ
φA − φB = ∫ABE·dl
通俗解释

电位像高度，正电荷会沿电位降低方向运动，所以电场方向是电位下降最快方向。

符号拆解
  • φ 是电位，单位 V。
  • ∇φ 指向电位增加最快方向。
  • 负号说明 E 指向电位降低方向。
  • E·dl 表示沿路径方向的电场贡献。
什么时候用

已知电位求电场时，用 E=−∇φ；求两点电位差时用线积分。

易错提醒

最常见错误是漏掉负号。

小题：φ=3x，求 E。
  1. 求梯度：∇φ=3eₓ。
  2. 用 E=−∇φ。
  3. 得到 E=−3eₓ。
答案整理：E=−3eₓ V/m。

2.4 真空中静电场方程：有源无旋

高斯定律无旋

静电场有两个基本性质：电荷是电场的源，所以散度与电荷密度有关；静电场无旋，所以可以引入电位。

∇·E = ρε₀
∇×E = 0
∯SE·dS = Qε₀
∮LE·dl = 0
通俗解释

∇·E 表示电场线发散程度；∇×E=0 表示静电场没有旋涡，沿闭合路径一圈做功为零。

符号拆解
  • ρ 是体电荷密度。
  • ε₀ 是真空介电常数。
  • Q 是闭合面内总电荷。
  • S 是闭合曲面，L 是闭合路径。
什么时候用

高斯定律适合对称电荷分布求 E；无旋性质用于引入电位。

易错提醒

高斯定律总成立，但不一定总方便求解。

小题：某处 ρ>0，∇·E 是正还是负？
  1. 根据 ∇·E=ρ/ε₀。
  2. ε₀ 为正。
  3. ρ>0，所以 ∇·E>0。
答案整理：∇·E 为正，说明该处电场线向外发散。

2.5 电介质中的 D：把自由电荷单独拿出来处理

电介质电位移

介质会被电场极化。引入 D 的目的，是在介质中更方便地处理自由电荷。

D = εE = εrε₀E
∇·D = ρf
∯SD·dS = Qf
通俗解释

E 描述电场强度；D 是为了处理介质中自由电荷而引入的辅助矢量。

符号拆解
  • D 是电位移矢量，单位 C/m²。
  • ε 是介电常数。
  • εᵣ 是相对介电常数。
  • ρ_f 是自由电荷密度。
什么时候用

介质中的高斯定律通常先求 D，再用 D=εE 求 E。

易错提醒

不要把自由电荷和束缚电荷混为一谈。

小题：线性介质中 D=4ε₀E，求 εᵣ。
  1. 线性介质关系：D=εᵣε₀E。
  2. 与题中 D=4ε₀E 比较。
  3. 得到 εᵣ=4。
答案整理：εᵣ=4。

2.6 导体静电平衡：内部 E=0，导体是等位体

导体等位体

导体中自由电荷能移动。静电平衡时如果内部仍有电场，电荷会继续移动，所以平衡条件是导体内部电场为零。

Einside = 0
φ = constant
通俗解释

E=−∇φ。若 E=0，则 ∇φ=0，说明导体内部电位不随位置变化，所以导体是等位体。

符号拆解
  • E_inside 是导体内部电场。
  • φ=constant 表示电位处处相同。
什么时候用

判断导体静电平衡、导体表面电荷分布时使用。

易错提醒

导体内部 E=0 不代表导体表面没有电荷，多余电荷通常在表面。

小题：静电平衡导体内部两点电位差是多少？
  1. 导体内部 E=0。
  2. 电位差与电场线积分有关。
  3. E=0，所以积分为 0。
答案整理：导体内部任意两点电位差为 0。

2.7 静电场边界条件：切向看 E，法向看 D

边界条件介质界面

场穿过不同介质分界面时，要分解成切向和法向。静电场中切向 E 连续，法向 D 的跳变由自由面电荷决定。

E1t = E2t
D2n − D1n = ρs
导体表面：Et=0，Dn=ρs
通俗解释

切向方向沿界面，法向方向垂直界面。边界条件就是告诉你场跨过界面时哪些分量不能乱跳。

符号拆解
  • t 表示切向。
  • n 表示法向。
  • ρ_s 是自由面电荷密度。
  • D=εE 可把 D 条件转成 E 条件。
什么时候用

介质分层、电场折射、导体表面电场题都要用。

易错提醒

不要把 E_t 连续误写成 D_t 连续；也不要把 D_n 条件直接当 E_n 连续。

小题：无自由面电荷时，D₁n 与 D₂n 关系？
  1. 边界条件：D₂n−D₁n=ρ_s。
  2. 无自由面电荷：ρ_s=0。
  3. 所以 D₂n−D₁n=0。
答案整理：D₁n = D₂n。

2.8 静电场能量：电场本身储存能量

能量电容

电场可以对电荷做功，所以电场本身具有能量。能量密度描述单位体积储存多少能量。

we = 12E·D
We = 12∭VE·D dV
W = 12CU2 = 12QU = Q22C
通俗解释

能量密度是局部量，总能量是把所有小体积里的能量加起来。电容器公式是常用特例。

符号拆解
  • w_e 是能量密度。
  • W_e 是总能量。
  • C 是电容。
  • U 是电压，Q 是电荷量。
什么时候用

计算电场储能、电容器能量时使用。

易错提醒

公式选择看题目给什么量：给 C/U 用 1/2CU²，给 Q/C 用 Q²/2C。

小题：C=2 μF，U=10 V，求电容器能量。
  1. 统一单位：C=2×10⁻⁶ F。
  2. 选公式 W=1/2CU²。
  3. 代入：W=1/2×2×10⁻⁶×10²。
  4. 计算：W=1×10⁻⁴ J。
答案整理：W=1.0×10⁻⁴ J。
