静电场边值问题

本章把第二章的电位 φ 作为主角，学习如何通过边界条件求空间电位。

本章独立学习路线

  1. 从 E 转向 φ用电位简化静电场求解。
  2. 判断方程有源用泊松，无源用拉普拉斯。
  3. 识别边界条件分清电位、法向导数和混合条件。
  4. 使用镜像法用等效电荷处理特殊边界。

3.1 静电场边值问题：已知边界，求空间电位

边值问题电位

边值问题就是：边界上的电位或电位导数已知，要求区域内部每一点的电位分布。

已知边界条件 → 求 φ(x,y,z) → 再由 E = −∇φ 求电场
通俗解释

它像知道房间墙壁温度，求房间内部温度分布；静电场中则是知道边界电位，求空间电位。

符号拆解
  • φ 是待求电位。
  • 边界条件是求解所需的约束。
  • E=−∇φ 用于由电位求电场。
什么时候用

遇到导体边界、电位已知、空间区域待求时使用边值问题思想。

易错提醒

边值问题不是只求某一点，而是求整个区域的函数分布。

小题：已知导体球表面电位，求球外电位属于什么问题？
  1. 题目给了边界上的电位。
  2. 要求空间区域中的电位。
  3. 这符合边值问题定义。
答案整理：属于静电场边值问题，具体是第一类边界条件问题。

3.2 泊松方程与拉普拉斯方程

微分方程电位

电位 φ 满足的方程取决于区域内有没有电荷。有电荷源时是泊松方程；无电荷区域是拉普拉斯方程。

泊松方程：∇²φ = −ρε
拉普拉斯方程：∇²φ = 0
通俗解释

ρ 是电荷源。如果区域里有电荷，电位会被源项驱动；如果没有电荷，电位只由边界决定。

符号拆解
  • ∇² 是拉普拉斯算子。
  • ρ 是体电荷密度。
  • ε 是介电常数。
  • φ 是电位。
什么时候用

判断区域有源或无源时选择对应方程。

易错提醒

不要把“空间里有电场”和“区域内有电荷”混淆。无源区也可以有电场。

小题：无电荷区域中电位满足什么方程？
  1. 无电荷说明 ρ=0。
  2. 泊松方程 ∇²φ=−ρ/ε。
  3. 代入 ρ=0，得到 ∇²φ=0。
答案整理：满足拉普拉斯方程 ∇²φ=0。

3.3 三类边界条件：狄里赫利、诺依曼、混合

边界条件分类

边界条件告诉你边界上已知什么。已知电位是第一类；已知法向导数是第二类；已知二者组合是第三类。

第一类：φ|S=f₁(r)
第二类：∂φ∂n|S=f₂(r)
第三类：aφ + b∂φ∂n = f₃(r)
通俗解释

第一类像告诉你海拔高度；第二类像告诉你坡度；第三类是高度和坡度的组合约束。

符号拆解
  • S 是边界。
  • ∂φ/∂n 是沿法线方向的变化率。
  • f₁、f₂、f₃ 是边界上给定函数。
  • a、b 是系数。
什么时候用

读题时先判断给的是电位、场的法向分量，还是二者组合。

易错提醒

法向导数和电场法向有关，因为 E_n=−∂φ/∂n。

小题：导体表面电位给定为 V₀，属于哪类边界条件？
  1. 题目直接给出边界电位。
  2. 第一类边界条件定义为 φ|S=f。
  3. 因此属于狄里赫利条件。
答案整理：属于第一类边界条件，也叫狄里赫利条件。

3.4 唯一性定理：解只要满足方程和边界，就不会错

唯一性理论基础

唯一性定理说明，在给定区域内，只要电位满足泊松/拉普拉斯方程，并满足边界条件，那么解就是唯一的。

方程 + 边界条件 → 唯一电位分布
通俗解释

这给镜像法等方法提供了底气：不管你怎么构造，只要边界和区域方程对，结果就是原问题的解。

符号拆解
  • 方程描述区域内部规律。
  • 边界条件描述区域边缘约束。
  • 唯一性说明不会存在两个不同正确解。
什么时候用

证明镜像法、判断构造解是否正确时使用。

易错提醒

唯一性定理不一定告诉你怎么找解，但告诉你找到后为什么可信。

小题：为什么镜像法只需检查边界条件？
  1. 镜像法构造的新电荷只放在求解区域外。
  2. 求解区域内仍满足原来的方程。
  3. 如果边界条件也相同，根据唯一性定理，区域内解相同。
答案整理：因为唯一性定理保证满足同一方程和同一边界条件的解唯一。

3.5 镜像法：用等效电荷代替导体边界影响

镜像法技巧

镜像法把复杂导体边界的作用，用边界外的等效电荷代替，使原问题变成无界空间中的电荷叠加问题。

原电荷 + 镜像电荷 → 满足边界条件的等效问题
通俗解释

镜像电荷不是真实存在的，它只是数学替身。它的作用是让导体表面电位条件被自动满足。

符号拆解
  • 原电荷是真实电荷。
  • 镜像电荷是构造的等效电荷。
  • 边界条件是镜像法成功的关键。
什么时候用

点电荷与无限大接地导体平面、导体球、导体圆柱等问题常用。

易错提醒

镜像电荷只能用于求解区域外或导体内部的数学构造，不能把它当真实电荷。

小题：点电荷 q 距接地无限导体平面 a，镜像电荷怎么放？
  1. 接地平面要求平面上电位为 0。
  2. 在平面对称位置放一个 −q。
  3. 原电荷和平面另一侧镜像电荷在平面上产生等大反号电位。
答案整理：在平面另一侧对称位置放镜像电荷 −q。
