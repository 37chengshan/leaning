第4章
恒定电流场
电流密度 J 理解电流不只是总量，还可以是空间分布。连续性 掌握 ∇·J=0 的物理意义。边界条件 学会跨介质处理 J 和 E。静电比拟 把静电场方法迁移到电流场。

学习路线

  • 电流密度 J 理解电流不只是总量，还可以是空间分布。
  • 连续性 掌握 ∇·J=0 的物理意义。
  • 边界条件 学会跨介质处理 J 和 E。
  • 静电比拟 把静电场方法迁移到电流场。

知识点

  • 4.1 恒定电流场：电荷持续定向运动形成电流
  • 4.2 恒定电流场方程：电流连续且无旋电场
  • 4.3 电流连续性原理：电流线必须闭合
  • 4.4 恒定电流场边界条件
  • 4.5 导电介质功率损耗
  • 4.6 静电比拟：恒定电场与静电场的相似性
  • 4.7 理想导电体中不存在恒定电场

公式体系

  • J = σE I = ∬ S J·dS
  • ∇·J = 0 ∯ S J·dS = 0 ∇×E = 0 ∮ L E·dl = 0
  • 稳态：∇·J = 0
  • J 1n = J 2n E 1t = E 2t
  • p = J·E = σE² = J² σ P = ∭ V p dV
  • σ ↔ ε J ↔ D G ↔ C G = (σ/ε)C
  • E = J σ ，当 σ→∞ 且 J 有限时，E→0

步骤题

  • 小题：σ=2 S/m，E=5 V/m，求 J。
  • 小题：闭合面内恒定电流净流出为多少？
  • 小题：为什么稳态导线中电流处处相等？
  • 小题：σ₁≠σ₂ 且 J₁n=J₂n，E₁n 与 E₂n 一定相等吗？
  • 小题：σ=4 S/m，E=3 V/m，求 p。
  • 小题：某结构 C=2 pF，σ/ε=10⁶ s⁻¹，求 G。
  • 小题：理想导体中若 J 有限，E 为多少？

章节联系

  • 第4章把静电场结构迁移到恒定电流场，形成 D=εE 与 J=σE 的类比。
