第4章｜恒定电流场

把静电场方法迁移到导电介质：E 推动电荷形成 J，稳态时电流连续。

5分钟预览

先记住本章一句话：

J=σE；∇·J=0；p=J·E；静电比拟：σ↔ε，J↔D，G↔C
J=σEI=∬J·dS∇·J=0边界条件功率损耗静电比拟

本章最容易错

  • 把 J 和 I 混淆
  • 忽略稳态 ∇·J=0
  • 不理解 σ↔ε 的比拟

第一优先级：概念

  • J=σE
  • I=∬J·dS
  • ∇·J=0

第二优先级：公式

  • J = σE I = ∬ S J·dS
  • ∇·J = 0 ∯ S J·dS = 0 ∇×E = 0 ∮ L E·dl = 0
  • 稳态：∇·J = 0
  • J 1n = J 2n E 1t = E 2t

第三优先级：训练

  • 小题：σ=2 S/m，E=5 V/m，求 J。
  • 小题：闭合面内恒定电流净流出为多少？
  • 小题：为什么稳态导线中电流处处相等？
  • 小题：σ₁≠σ₂ 且 J₁n=J₂n，E₁n 与 E₂n 一定相等吗？
