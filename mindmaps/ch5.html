第5章
恒定磁场
从电流到磁场 理解恒定电流是磁场旋度的来源。B 与 H 区分磁感应强度和磁场强度。磁化 理解材料内部磁矩排列造成的宏观效应。边界和转矩 掌握磁介质边界条件与线圈受力矩。

学习路线

  • 从电流到磁场 理解恒定电流是磁场旋度的来源。
  • B 与 H 区分磁感应强度和磁场强度。
  • 磁化 理解材料内部磁矩排列造成的宏观效应。
  • 边界和转矩 掌握磁介质边界条件与线圈受力矩。

知识点

  • 5.1 真空中恒定磁场方程：有旋无散
  • 5.2 介质中的恒定磁场方程：用 H 处理传导电流
  • 5.3 磁介质本构关系：B、H、M 的联系
  • 5.4 磁化电流：磁化也等效为电流
  • 5.5 恒定磁场边界条件
  • 5.6 理想导磁体：μ→∞ 时 H 趋于 0
  • 5.7 磁矩与转矩：载流线圈在磁场中会转动

公式体系

  • ∇×B = μ₀J ∇·B = 0 ∮ L B·dl = μ₀I
  • ∇×H = J ∇·B = 0 ∮ L H·dl = I
  • B = μ₀(H + M) 线性各向同性介质：B = μH = μ r μ₀H
  • J m = ∇×M K m = M×n
  • B 1n = B 2n H 2t − H 1t = K s
  • H = B μ ，当 μ→∞ 且 B 有限时，H→0
  • m = NIS n T = m×B |T| = mB sinθ

步骤题

  • 小题：为什么 ∇·B=0 表示磁力线闭合？
  • 小题：若闭合路径包围电流 5 A，则 ∮H·dl 等于多少？
  • 小题：μᵣ=100，H=10 A/m，求 B 大小。
  • 小题：均匀 M 且无空间变化时，J_m 如何？
  • 小题：无面电流时 H₁t 与 H₂t 关系？
  • 小题：μ→∞ 且 B 有限，H 为多少？
  • 小题：N=100，I=2A，S=0.02m²，B=0.5T，θ=60°，求转矩。

章节联系

  • 第5章从电流进入磁场，建立 B、H、磁介质和边界条件。
